## 附录与排版约定

- **代码块**：注明语言；配置与 Java 类名用反引号标出。
- **图示**：复杂流程可放一张 mermaid `flowchart`（节点 ID 勿用空格）；与具体产品截图二选一即可。
- **字数**：成稿以 **3000～5000 字**（中文）为目标；代码与命令计入篇幅时，优先压缩重复说明。
