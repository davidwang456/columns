# 专栏正文（第 1–36 章）

文件命名：`第 {两位章号} 章 {主题}.md`（章号与主题之间有空格），主题与章标题一致，并与 [CHAPTER_INDEX.md](../CHAPTER_INDEX.md) 中条目 ID（`ch-NN`）、英文 Slug 对齐。

每章结构：专栏主干（**1 项目背景 → 2 项目设计：大师与小白 → 3 项目实战：主代码片段 → 4 项目总结**）+ **多波段主题拓展**（仍围绕同一主题对象，按波段延伸背景、设计、实战与总结向内容）。

汉字体量说明：经 `columns/scripts/append_depth_sections*.py` 系列脚本与主干合并后，多数章节总汉字数在 **约 3800–5000+** 区间（随主干长短浮动）。若用于正式出版或内训，可按品牌规范裁剪主题拓展波段。
