# Three.js 实战专栏总纲（38章）

本专栏以“趣味性 + 实战性 + 深度”作为统一标准，覆盖从入门到生产级落地的完整能力路径。

## 分级说明

- 基础篇（13 章）：核心概念、单机版、简单 API、初级实战（新人开发、测试）
- 中级篇（13 章）：架构设计、分布式场景、性能调优、可观测性（核心开发、运维）
- 高级篇（12 章）：源码剖析、极端优化、自定义扩展、SRE 落地（架构师、资深开发）

## 章节目录

### 基础篇（13章）

1. Three.js 全景认知与首个业务级 3D 看板
2. 坐标系、相机与视角控制：从“看得见”到“看得准”
3. 几何体与网格：搭建仓储数字孪生的基础积木
4. 材质系统入门：PBR 与业务可视化质感表达
5. 光照系统与阴影：让产线场景“可信”而非“玩具”
6. 纹理贴图与资源管理：大屏素材规范化加载
7. 动画基础：关键帧、缓动与设备状态动效
8. 交互入门：射线拾取与对象选择（告警点选）
9. UI 叠加与 2D/3D 联动：图表与三维场景协同
10. 相机漫游与路径导览：园区巡检导览系统
11. 模型导入（glTF）与资产流程：美术到前端协作链路
12. 调试与开发效率：坐标辅助、性能面板、开发规范
13. 基础综合项目：单机场景版“智慧工厂驾驶舱”

### 中级篇（13章）

14. 场景架构分层：渲染层、业务层、状态层解耦
15. 模块化工程实践：Vite + TypeScript + Three.js 目录设计
16. 海量对象渲染优化：Instancing 与合批策略
17. LOD 与分块加载：超大园区流畅渲染方案
18. 资源生命周期管理：内存泄漏排查与自动回收
19. 后处理管线：Bloom、SSAO、Outline 在业务中的取舍
20. WebGL 性能调优：CPU/GPU 瓶颈定位方法论
21. 实时数据驱动：WebSocket 推流与场景状态同步
22. 多端适配：大屏、PC、平板分辨率与交互策略
23. 可观测性建设：帧率、卡顿、崩溃指标采集
24. 三维权限与安全边界：数据脱敏与模型访问控制
25. 测试体系：可视回归、交互回放、关键路径自动化
26. 中级综合项目：多场景联动的“城市运行中心”

### 高级篇（12章）

27. Three.js 渲染主流程源码导读
28. BufferGeometry/Material 核心机制深潜
29. Shader 入门到实战：GLSL 定制业务特效
30. 后处理源码与自定义 Pass 设计
31. 复杂特效工程化：体积雾、流光、热力场
32. 大规模地图与 GIS 融合（Tiles/地理坐标转换）
33. 极限性能策略：百万级对象与分帧更新
34. Web Worker 与 OffscreenCanvas 并行渲染实践
35. 端到端稳定性：错误隔离、降级与熔断策略
36. SRE 落地：容量评估、压测、发布回滚与值班手册
37. 自定义扩展：插件体系、场景编辑器、组件市场化
38. 高级综合项目：生产级“数字孪生运营平台”

## 文件结构

- `basic/`：基础篇独立章节文件
- `intermediate/`：中级篇独立章节文件
- `advanced/`：高级篇独立章节文件
- `template.md`：每章统一写作模板

## 写作硬性约束

- 每章独立文件，目标字数 3000-5000
- 固定四段式结构：项目背景、项目设计（角色对话）、项目实战、项目总结
- 所有实战代码需可运行，附验证步骤
- 每章末尾包含跨部门协作建议与思考题
