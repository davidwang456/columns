# 7.2 句法模式识别的Java实现

> **第7章 结构模式识别** | 结构聚类算法 · 句法模式识别

## 背景

句法模式识别（Syntactic Pattern Recognition）是结构模式识别的重要分支，由K.S. Fu在20世纪70年代系统发展。与统计模式识别使用数值特征向量描述模式不同，句法模式识别用符号串（或图、树等结构）来描述模式，用形式语言和文法来定义模式的类别。

句法模式识别的核心思想：
1. **基元提取**：将复杂模式分解为基本的不可再分的子模式（基元/原语）
2. **文法定义**：为每种模式类别定义一套产生规则（文法），描述基元的合法组合方式
3. **句法分析**：对待识别的模式提取基元串后，用语法分析器判断该串是否属于某个文法定义的语言

形式文法 G = (V_N, V_T, P, S)：
- V_N：非终结符集合
- V_T：终结符集合（基元）
- P：产生规则集合
- S：起始符号

例如，识别简单几何图形：
- 三角形 = 三条线段顺序连接且首尾相连
- 矩形 = 四条线段交替水平和垂直连接

## Chomsky层次结构

Noam Chomsky在1956年提出的文法层次结构，将形式文法分为四个层级（从弱到强）：

**3型文法（正则文法，Regular Grammar）**：
- 产生规则形式：A → aB 或 A → a（右线性），或 A → Ba 或 A → a（左线性）
- 对应的自动机：有限状态自动机（FSA / DFA / NFA）
- 生成的语言：正则语言，可用正则表达式描述
- 能力：能描述具有固定重复模式的简单结构，如 "a*b+" 表示若干个a后跟若干个b
- 局限：无法处理嵌套、配对等结构（如匹配括号）

**2型文法（上下文无关文法，CFG）**：
- 产生规则形式：A → α，其中 A 是单个非终结符，α 是任意符号串
- 对应的自动机：下推自动机（PDA）
- 生成的语言：上下文无关语言
- 能力：能描述嵌套结构、递归结构，如匹配括号 {aⁿbⁿ | n≥1}
- 典型应用：编程语言语法、算术表达式、XML文档结构

**1型文法（上下文相关文法，CSG）**：
- 产生规则形式：αAβ → αγβ，其中 |γ| ≥ 1
- 对应的自动机：线性有界自动机（LBA）
- 生成的语言：上下文相关语言
- 能力：能描述依赖上下文的结构，如 {aⁿbⁿcⁿ | n≥1}

**0型文法（无限制文法，Unrestricted Grammar）**：
- 产生规则形式：α → β，无限制
- 对应的自动机：图灵机
- 生成的语言：递归可枚举语言

在模式识别中，2型文法（CFG）是最常用的，它在表达能力和计算效率之间取得了良好平衡。

## 文法类型与自动机的对应关系

每种Chomsky文法类型都有一个等价的自动机模型，这一对应关系是形式语言理论的核心定理：

| 文法类型 | 自动机模型 | 识别算法复杂度 |
|---------|-----------|--------------|
| 3型（正则） | 有限状态自动机 | O(n)，线性扫描 |
| 2型（上下文无关） | 下推自动机 | O(n³)，CYK/Earley算法 |
| 1型（上下文相关） | 线性有界自动机 | PSPACE完全 |
| 0型（无限制） | 图灵机 | 不可判定 |

在句法模式识别中，选择哪种文法取决于模式的复杂度。简单的序列模式（如DNA重复序列）用正则文法即可；具有嵌套或递归结构的模式（如括号匹配、树形结构）需要上下文无关文法；而某些复杂的依赖关系（如交叉引用）则可能需要上下文相关文法。

## 句法方法的实际局限性

尽管句法方法理论优美，但在实际应用中面临若干挑战：

1. **基元提取的可靠性**：从原始信号（图像、语音）中提取离散基元本身就是一个困难的模式识别问题。提取错误直接导致后续解析失败。
2. **文法的人工设计**：需要领域专家根据先验知识设计文法规则，自动文法推断虽有研究但尚不成熟。
3. **噪声敏感性**：严格的语法匹配无法容忍基元级别的错误，需要引入错误容忍机制。
4. **计算开销**：通用的CFG解析算法（CYK、Earley）时间复杂度为 O(n³)，对长序列不够高效。

现代做法通常将句法方法与统计方法结合——用统计模型（HMM、神经网络）提取基元，再用句法框架进行结构分析。

句法模式识别广泛应用于：
- 字符/手写体识别
- 染色体形状分类
- 电路图识别
- 编程语言语法检查

## 句法模式识别的java实现

下面实现一个完整的句法模式识别系统，包括文法定义、CYK语法分析器和模式分类：

```
import java.util.*;

public class SyntacticPatternRecognition {

    /**
     * 上下文无关文法(CFG)
     */
    static class Grammar {
        String name;
        Set<String> nonTerminals = new HashSet<>();
        Set<String> terminals = new HashSet<>();
        List<String[]> rules = new ArrayList<>();  // [左部, 右部1, 右部2...] 
        String startSymbol;

        Grammar(String name, String startSymbol) {
            this.name = name;
            this.startSymbol = startSymbol;
            nonTerminals.add(startSymbol);
        }

        // 添加产生规则: A -> B C 或 A -> a
        void addRule(String lhs, String... rhs) {
            String[] rule = new String[rhs.length + 1];
            rule[0] = lhs;
            System.arraycopy(rhs, 0, rule, 1, rhs.length);
            rules.add(rule);
            nonTerminals.add(lhs);
            for (String s : rhs) {
                if (s.equals(s.toLowerCase()) && s.length() == 1) terminals.add(s);
                else nonTerminals.add(s);
            }
        }

        @Override
        public String toString() {
            StringBuilder sb = new StringBuilder();
            sb.append("文法: ").append(name).append("\n");
            sb.append("产生规则:\n");
            for (String[] rule : rules) {
                sb.append("  ").append(rule[0]).append(" -> ");
                for (int i = 1; i < rule.length; i++) {
                    if (i > 1) sb.append(" ");
                    sb.append(rule[i]);
                }
                sb.append("\n");
            }
            return sb.toString();
        }
    }

    /**
     * CYK算法（Cocke-Younger-Kasami）
     * 用于判断字符串是否能被CNF文法生成
     */
    static class CYKParser {
        Grammar grammar;

        CYKParser(Grammar grammar) {
            this.grammar = grammar;
        }

        // 判断输入串是否属于该文法定义的语言
        boolean parse(String[] input) {
            int n = input.length;
            if (n == 0) return false;

            // table[i][j] 存储能产生 input[i..j] 的非终结符集合
            @SuppressWarnings("unchecked")
            Set<String>[][] table = new HashSet[n][n];
            for (int i = 0; i < n; i++)
                for (int j = 0; j < n; j++)
                    table[i][j] = new HashSet<>();

            // 填充对角线：单个终结符
            for (int i = 0; i < n; i++) {
                for (String[] rule : grammar.rules) {
                    if (rule.length == 2 && rule[1].equals(input[i]))
                        table[i][i].add(rule[0]);
                }
            }

            // 自底向上填表
            for (int len = 2; len <= n; len++) {
                for (int i = 0; i <= n - len; i++) {
                    int j = i + len - 1;
                    for (int k = i; k < j; k++) {
                        for (String[] rule : grammar.rules) {
                            if (rule.length == 3) {
                                if (table[i][k].contains(rule[1]) && table[k + 1][j].contains(rule[2]))
                                    table[i][j].add(rule[0]);
                            }
                        }
                    }
                }
            }

            return table[0][n - 1].contains(grammar.startSymbol);
        }
    }

    /**
     * 简单的递归下降解析器（用于更直观的文法）
     */
    static class RecursiveDescentParser {
        Grammar grammar;
        String[] input;
        int pos;

        RecursiveDescentParser(Grammar grammar) {
            this.grammar = grammar;
        }

        boolean parse(String[] input) {
            this.input = input;
            this.pos = 0;
            boolean result = matchSymbol(grammar.startSymbol);
            return result && pos == input.length;
        }

        private boolean matchSymbol(String symbol) {
            // 如果是终结符
            if (grammar.terminals.contains(symbol)) {
                if (pos < input.length && input[pos].equals(symbol)) {
                    pos++;
                    return true;
                }
                return false;
            }

            // 如果是非终结符，尝试所有规则
            int savedPos = pos;
            for (String[] rule : grammar.rules) {
                if (rule[0].equals(symbol)) {
                    pos = savedPos;
                    boolean match = true;
                    for (int i = 1; i < rule.length; i++) {
                        if (!matchSymbol(rule[i])) { match = false; break; }
                    }
                    if (match) return true;
                }
            }
            pos = savedPos;
            return false;
        }
    }

    /**
     * 模式分类器：用多个文法定义不同的模式类别
     */
    static class PatternClassifier {
        Map<String, Grammar> classGrammars = new LinkedHashMap<>();
        Map<String, CYKParser> parsers = new LinkedHashMap<>();

        void addClass(String className, Grammar grammar) {
            classGrammars.put(className, grammar);
            parsers.put(className, new CYKParser(grammar));
        }

        String classify(String[] input) {
            for (Map.Entry<String, CYKParser> entry : parsers.entrySet()) {
                if (entry.getValue().parse(input))
                    return entry.getKey();
            }
            return "未知模式";
        }

        List<String> classifyAll(String[] input) {
            List<String> matches = new ArrayList<>();
            for (Map.Entry<String, CYKParser> entry : parsers.entrySet()) {
                if (entry.getValue().parse(input))
                    matches.add(entry.getKey());
            }
            return matches;
        }
    }

    public static void main(String[] args) {
        System.out.println("=== 句法模式识别 ===\n");

        // 示例1：几何图形识别
        // 基元：h(水平线), v(垂直线), d(斜线)
        System.out.println("--- 示例1：几何图形识别 ---\n");

        // 三角形文法（CNF）：三条斜线连接
        // S -> D R, R -> D D, D -> d
        Grammar triangleGrammar = new Grammar("三角形", "S");
        triangleGrammar.addRule("S", "D", "R");
        triangleGrammar.addRule("R", "D", "D");
        triangleGrammar.addRule("D", "d");

        // 矩形文法（CNF）：h v h v
        // S -> H T, T -> V U, U -> H V, H -> h, V -> v
        Grammar rectangleGrammar = new Grammar("矩形", "S");
        rectangleGrammar.addRule("S", "H", "T");
        rectangleGrammar.addRule("T", "V", "U");
        rectangleGrammar.addRule("U", "H", "V");
        rectangleGrammar.addRule("H", "h");
        rectangleGrammar.addRule("V", "v");

        // 直线文法：单个h或v
        // S -> h | v
        Grammar lineGrammar = new Grammar("直线", "S");
        lineGrammar.addRule("S", "h");
        lineGrammar.addRule("S", "v");

        System.out.println(triangleGrammar);
        System.out.println(rectangleGrammar);

        PatternClassifier shapeClassifier = new PatternClassifier();
        shapeClassifier.addClass("三角形", triangleGrammar);
        shapeClassifier.addClass("矩形", rectangleGrammar);
        shapeClassifier.addClass("直线", lineGrammar);

        String[][] shapeTests = {
            {"d", "d", "d"},     // 三角形
            {"h", "v", "h", "v"}, // 矩形
            {"h"},                // 直线
            {"d", "d"},           // 未知
            {"v"},                // 直线
        };

        for (String[] pattern : shapeTests) {
            String result = shapeClassifier.classify(pattern);
            System.out.println("基元串 " + Arrays.toString(pattern) + " -> " + result);
        }

        // 示例2：简单算术表达式的语法检查
        System.out.println("\n--- 示例2：表达式模式识别 ---\n");

        // 简化的表达式文法(CNF): n + n, n * n, (n + n) 等
        // 用基元: n(数字), p(加号), m(乘号), l(左括号), r(右括号)
        Grammar exprGrammar = new Grammar("算术表达式", "S");
        exprGrammar.addRule("S", "N");             // S -> N (单个数)
        exprGrammar.addRule("S", "N", "T");        // S -> N T (数+运算符尾)
        exprGrammar.addRule("T", "P", "N");        // T -> P N (+ n)
        exprGrammar.addRule("T", "M", "N");        // T -> M N (* n)
        exprGrammar.addRule("N", "n");
        exprGrammar.addRule("P", "p");
        exprGrammar.addRule("M", "m");

        CYKParser exprParser = new CYKParser(exprGrammar);

        String[][] exprTests = {
            {"n"},               // 合法：单个数
            {"n", "p", "n"},     // 合法：n + n
            {"n", "m", "n"},     // 合法：n * n
            {"p", "n"},          // 非法：+ n
            {"n", "n"},          // 非法：n n
        };

        System.out.println("表达式语法检查:");
        for (String[] expr : exprTests) {
            boolean valid = exprParser.parse(expr);
            System.out.println("  " + Arrays.toString(expr) + " -> " + (valid ? "合法表达式" : "非法表达式"));
        }

        // 示例3：DNA序列模式识别
        System.out.println("\n--- 示例3：DNA序列模式 ---\n");

        // 简化的回文序列文法（如限制性酶切位点）
        // 模式: a...t 配对, c...g 配对 -> 回文序列
        Grammar palindromeGrammar = new Grammar("回文序列", "S");
        palindromeGrammar.addRule("S", "A", "X");  // a...t
        palindromeGrammar.addRule("X", "S", "T");
        palindromeGrammar.addRule("S", "C", "Y");  // c...g
        palindromeGrammar.addRule("Y", "S", "G");
        palindromeGrammar.addRule("S", "A", "T");  // at
        palindromeGrammar.addRule("S", "C", "G");  // cg
        palindromeGrammar.addRule("A", "a");
        palindromeGrammar.addRule("T", "t");
        palindromeGrammar.addRule("C", "c");
        palindromeGrammar.addRule("G", "g");

        CYKParser dnaParser = new CYKParser(palindromeGrammar);
        String[][] dnaTests = {
            {"a", "t"},                 // at 回文
            {"a", "c", "g", "t"},       // acgt 回文
            {"c", "a", "t", "g"},       // catg 回文
            {"a", "a", "t"},            // 非回文
        };

        System.out.println("DNA回文序列检测:");
        for (String[] seq : dnaTests) {
            boolean isPalindrome = dnaParser.parse(seq);
            System.out.println("  " + Arrays.toString(seq) + " -> " + (isPalindrome ? "回文序列" : "非回文序列"));
        }
    }
}

下面实现Earley解析器，它能处理任意上下文无关文法（包括歧义文法），不要求文法转换为Chomsky范式：

```
import java.util.*;

public class EarleyParser {

    static class Rule {
        String lhs;
        String[] rhs;
        Rule(String lhs, String... rhs) { this.lhs = lhs; this.rhs = rhs; }
        @Override
        public String toString() { return lhs + " -> " + String.join(" ", rhs); }
    }

    static class EarleyItem {
        Rule rule;
        int dot;
        int origin;

        EarleyItem(Rule rule, int dot, int origin) {
            this.rule = rule; this.dot = dot; this.origin = origin;
        }

        boolean isComplete() { return dot >= rule.rhs.length; }

        String nextSymbol() {
            return (dot < rule.rhs.length) ? rule.rhs[dot] : null;
        }

        @Override
        public boolean equals(Object o) {
            if (!(o instanceof EarleyItem)) return false;
            EarleyItem other = (EarleyItem) o;
            return rule == other.rule && dot == other.dot && origin == other.origin;
        }

        @Override
        public int hashCode() {
            return Objects.hash(System.identityHashCode(rule), dot, origin);
        }

        @Override
        public String toString() {
            StringBuilder sb = new StringBuilder(rule.lhs + " -> ");
            for (int i = 0; i < rule.rhs.length; i++) {
                if (i == dot) sb.append("• ");
                sb.append(rule.rhs[i] + " ");
            }
            if (dot == rule.rhs.length) sb.append("•");
            sb.append(" [" + origin + "]");
            return sb.toString();
        }
    }

    private List<Rule> rules;
    private Set<String> nonTerminals;
    private Set<String> terminals;
    private String startSymbol;

    public EarleyParser(String startSymbol) {
        this.rules = new ArrayList<>();
        this.nonTerminals = new HashSet<>();
        this.terminals = new HashSet<>();
        this.startSymbol = startSymbol;
        nonTerminals.add(startSymbol);
    }

    public void addRule(String lhs, String... rhs) {
        rules.add(new Rule(lhs, rhs));
        nonTerminals.add(lhs);
        for (String s : rhs) {
            if (s.length() == 1 && Character.isLowerCase(s.charAt(0)))
                terminals.add(s);
            else
                nonTerminals.add(s);
        }
    }

    @SuppressWarnings("unchecked")
    public boolean parse(String[] input, boolean verbose) {
        int n = input.length;
        Set<EarleyItem>[] chart = new LinkedHashSet[n + 1];
        for (int i = 0; i <= n; i++) chart[i] = new LinkedHashSet<>();

        Rule augmented = new Rule("S'", startSymbol);
        chart[0].add(new EarleyItem(augmented, 0, 0));

        for (int i = 0; i <= n; i++) {
            Queue<EarleyItem> worklist = new LinkedList<>(chart[i]);
            Set<EarleyItem> processed = new HashSet<>(chart[i]);

            while (!worklist.isEmpty()) {
                EarleyItem item = worklist.poll();

                if (item.isComplete()) {
                    for (EarleyItem originItem : new ArrayList<>(chart[item.origin])) {
                        if (!originItem.isComplete() && item.rule.lhs.equals(originItem.nextSymbol())) {
                            EarleyItem newItem = new EarleyItem(originItem.rule, originItem.dot + 1, originItem.origin);
                            if (chart[i].add(newItem) && processed.add(newItem))
                                worklist.add(newItem);
                        }
                    }
                } else {
                    String next = item.nextSymbol();
                    if (nonTerminals.contains(next)) {
                        for (Rule r : rules) {
                            if (r.lhs.equals(next)) {
                                EarleyItem newItem = new EarleyItem(r, 0, i);
                                if (chart[i].add(newItem) && processed.add(newItem))
                                    worklist.add(newItem);
                            }
                        }
                    } else if (i < n && next.equals(input[i])) {
                        EarleyItem newItem = new EarleyItem(item.rule, item.dot + 1, item.origin);
                        chart[i + 1].add(newItem);
                    }
                }
            }

            if (verbose) {
                System.out.println("  S(" + i + "): ");
                for (EarleyItem item : chart[i])
                    System.out.println("    " + item);
            }
        }

        for (EarleyItem item : chart[n]) {
            if (item.rule.lhs.equals("S'") && item.isComplete() && item.origin == 0)
                return true;
        }
        return false;
    }

    public static void main(String[] args) {
        System.out.println("=== Earley解析器 ===\n");

        System.out.println("--- 示例1：歧义文法解析 ---\n");

        EarleyParser ambiguous = new EarleyParser("E");
        ambiguous.addRule("E", "E", "p", "E");
        ambiguous.addRule("E", "E", "m", "E");
        ambiguous.addRule("E", "n");

        String[][] ambTests = {
            {"n"},
            {"n", "p", "n"},
            {"n", "p", "n", "m", "n"},
            {"n", "p", "n", "p", "n", "m", "n"},
            {"p", "n"},
            {"n", "n"},
        };

        for (String[] test : ambTests) {
            boolean result = ambiguous.parse(test, false);
            System.out.println("  " + Arrays.toString(test) + " -> " + (result ? "接受" : "拒绝"));
        }

        System.out.println("\n详细解析过程 (n + n * n):");
        ambiguous.parse(new String[]{"n", "p", "n", "m", "n"}, true);

        System.out.println("\n--- 示例2：括号配对文法 ---\n");
        System.out.println("文法: S -> ( S ) | S S | ε\n");

        EarleyParser paren = new EarleyParser("S");
        paren.addRule("S", "l", "S", "r");
        paren.addRule("S", "S", "S");
        paren.addRule("S", "l", "r");

        String[][] parenTests = {
            {"l", "r"},
            {"l", "l", "r", "r"},
            {"l", "r", "l", "r"},
            {"l", "l", "r", "l", "r", "r"},
            {"l", "l", "r"},
            {"r", "l"},
        };

        for (String[] test : parenTests) {
            String display = String.join("", test).replace("l", "(").replace("r", ")");
            boolean result = paren.parse(test, false);
            System.out.println("  " + display + " -> " + (result ? "括号匹配" : "括号不匹配"));
        }

        System.out.println("\n--- 示例3：自然语言短语结构 ---\n");
        System.out.println("文法: S -> NP VP, NP -> D N, VP -> V NP | V\n");

        EarleyParser nlp = new EarleyParser("S");
        nlp.addRule("S", "NP", "VP");
        nlp.addRule("NP", "D", "N");
        nlp.addRule("VP", "V", "NP");
        nlp.addRule("VP", "V");
        nlp.nonTerminals.addAll(Arrays.asList("NP", "VP", "D", "N", "V"));
        nlp.terminals.addAll(Arrays.asList("d", "n", "v"));
        nlp.addRule("D", "d");
        nlp.addRule("N", "n");
        nlp.addRule("V", "v");

        String[][] nlpTests = {
            {"d", "n", "v"},

        String[] nlpDesc = {"定语+名词+动词（如:那只猫跑）", "定语+名词+动词+定语+名词（如:那只猫追那只鼠）",

下面实现一个手写笔画识别的实际应用——将手写字符的笔画序列（垂直、水平、斜线、曲线）作为基元，通过句法分析识别手写字符：

```
import java.util.*;

public class HandwritingStrokeRecognition {

    static class StrokeGrammar {
        String charName;
        String startSymbol;
        List<String[]> rules = new ArrayList<>();
        Set<String> nonTerminals = new HashSet<>();
        Set<String> terminals = new HashSet<>(Arrays.asList("v", "h", "d", "u", "c", "r"));

        StrokeGrammar(String charName, String startSymbol) {
            this.charName = charName;
            this.startSymbol = startSymbol;
            nonTerminals.add(startSymbol);
        }

        void addRule(String lhs, String... rhs) {
            String[] rule = new String[rhs.length + 1];
            rule[0] = lhs;
            System.arraycopy(rhs, 0, rule, 1, rhs.length);
            rules.add(rule);
            nonTerminals.add(lhs);
        }

        @SuppressWarnings("unchecked")
        boolean parse(String[] input) {
            int n = input.length;
            if (n == 0) return false;
            Set<String>[][] table = new HashSet[n][n];
            for (int i = 0; i < n; i++)
                for (int j = 0; j < n; j++) table[i][j] = new HashSet<>();

            for (int i = 0; i < n; i++)
                for (String[] rule : rules)
                    if (rule.length == 2 && rule[1].equals(input[i]))
                        table[i][i].add(rule[0]);

            for (int len = 2; len <= n; len++)
                for (int i = 0; i <= n - len; i++) {
                    int j = i + len - 1;
                    for (int k = i; k < j; k++)
                        for (String[] rule : rules)
                            if (rule.length == 3 && table[i][k].contains(rule[1]) && table[k + 1][j].contains(rule[2]))
                                table[i][j].add(rule[0]);
                }
            return table[0][n - 1].contains(startSymbol);
        }
    }

    static Map<String, StrokeGrammar> buildCharGrammars() {
        Map<String, StrokeGrammar> grammars = new LinkedHashMap<>();

        StrokeGrammar gT = new StrokeGrammar("T", "S");
        gT.addRule("S", "H", "V1");
        gT.addRule("H", "h");
        gT.addRule("V1", "v");
        grammars.put("T", gT);

        StrokeGrammar gL = new StrokeGrammar("L", "S");
        gL.addRule("S", "V1", "H");
        gL.addRule("V1", "v");
        gL.addRule("H", "h");
        grammars.put("L", gL);

        StrokeGrammar gPlus = new StrokeGrammar("+", "S");
        gPlus.addRule("S", "H", "V1");
        gPlus.addRule("S", "V1", "H");
        gPlus.addRule("H", "h");
        gPlus.addRule("V1", "v");
        grammars.put("+", gPlus);

        StrokeGrammar gX = new StrokeGrammar("X", "S");
        gX.addRule("S", "D1", "D2");
        gX.addRule("D1", "d");
        gX.addRule("D2", "u");
        grammars.put("X", gX);

        StrokeGrammar gZ = new StrokeGrammar("Z", "S");
        gZ.addRule("S", "H", "R");
        gZ.addRule("R", "D1", "H");
        gZ.addRule("H", "h");
        gZ.addRule("D1", "d");
        grammars.put("Z", gZ);

        StrokeGrammar gN = new StrokeGrammar("N", "S");
        gN.addRule("S", "V1", "R");
        gN.addRule("R", "D1", "V2");
        gN.addRule("V1", "v");
        gN.addRule("D1", "d");
        gN.addRule("V2", "v");
        grammars.put("N", gN);

        StrokeGrammar gF = new StrokeGrammar("F", "S");
        gF.addRule("S", "V1", "R");
        gF.addRule("R", "H", "H2");
        gF.addRule("V1", "v");
        gF.addRule("H", "h");
        gF.addRule("H2", "h");
        grammars.put("F", gF);

        StrokeGrammar gD = new StrokeGrammar("D", "S");
        gD.addRule("S", "V1", "C1");
        gD.addRule("V1", "v");
        gD.addRule("C1", "c");
        grammars.put("D", gD);

        return grammars;
    }

    static String recognize(Map<String, StrokeGrammar> grammars, String[] strokes) {
        List<String> matches = new ArrayList<>();
        for (Map.Entry<String, StrokeGrammar> entry : grammars.entrySet()) {
            if (entry.getValue().parse(strokes))
                matches.add(entry.getKey());
        }
        if (matches.isEmpty()) return "无法识别";
        if (matches.size() == 1) return matches.get(0);
        return matches.toString() + " (歧义)";
    }

    public static void main(String[] args) {
        System.out.println("=== 手写笔画识别 ===\n");
        System.out.println("基元定义:");
        System.out.println("  v = 垂直笔画 (vertical)");
        System.out.println("  h = 水平笔画 (horizontal)");
        System.out.println("  d = 斜线笔画-右下 (diagonal)");
        System.out.println("  u = 斜线笔画-右上 (up-diagonal)");
        System.out.println("  c = 曲线笔画 (curve)");
        System.out.println("  r = 右转弯 (right-turn)\n");

        Map<String, StrokeGrammar> grammars = buildCharGrammars();

        System.out.println("已注册的字符文法:");
        for (Map.Entry<String, StrokeGrammar> entry : grammars.entrySet()) {
            StrokeGrammar g = entry.getValue();
            System.out.print("  字符 '" + entry.getKey() + "': ");
            for (String[] rule : g.rules) {
                System.out.print(rule[0] + "->");
                for (int i = 1; i < rule.length; i++) System.out.print(rule[i]);
                System.out.print("  ");
            }
            System.out.println();
        }

        System.out.println("\n笔画序列识别:");
        String[][] testStrokes = {
            {"h", "v"},         // T 或 +
            {"v", "h"},         // L 或 +
            {"d", "u"},         // X
            {"h", "d", "h"},    // Z
            {"v", "d", "v"},    // N
            {"v", "h", "h"},    // F
            {"v", "c"},         // D
            {"h", "h"},         // ?
            {"v", "v", "v"},    // ?
            {"d", "d"},         // ?
        };

        for (String[] strokes : testStrokes) {
            String result = recognize(grammars, strokes);
            System.out.println("  笔画 " + Arrays.toString(strokes) + " -> 识别为: " + result);
        }

        System.out.println("\n歧义分析:");
        System.out.println("  'h v' 可识别为 T（先横后竖）或 +（先横后竖），");
        System.out.println("  实际应用中需结合笔画位置关系（交叉点、端点连接方式）消歧。");
    }
}

下面实现容错解析器——允许输入序列中存在一个符号错误（插入、删除或替换），通过编辑距离思想实现鲁棒的句法识别：

```
import java.util.*;

public class ErrorTolerantParser {

    static class Grammar {
        String name;
        String startSymbol;
        Set<String> nonTerminals = new HashSet<>();
        Set<String> terminals = new HashSet<>();
        List<String[]> rules = new ArrayList<>();

        Grammar(String name, String startSymbol) {
            this.name = name;
            this.startSymbol = startSymbol;
            nonTerminals.add(startSymbol);
        }

        void addRule(String lhs, String... rhs) {
            String[] rule = new String[rhs.length + 1];
            rule[0] = lhs;
            System.arraycopy(rhs, 0, rule, 1, rhs.length);
            rules.add(rule);
            nonTerminals.add(lhs);
            for (String s : rhs)
                if (s.length() == 1 && Character.isLowerCase(s.charAt(0))) terminals.add(s);
                else nonTerminals.add(s);
        }

        @SuppressWarnings("unchecked")
        boolean exactParse(String[] input) {
            int n = input.length;
            if (n == 0) return false;
            Set<String>[][] table = new HashSet[n][n];
            for (int i = 0; i < n; i++)
                for (int j = 0; j < n; j++) table[i][j] = new HashSet<>();

            for (int i = 0; i < n; i++)
                for (String[] rule : rules)
                    if (rule.length == 2 && rule[1].equals(input[i])) table[i][i].add(rule[0]);

            for (int len = 2; len <= n; len++)
                for (int i = 0; i <= n - len; i++) {
                    int j = i + len - 1;
                    for (int k = i; k < j; k++)
                        for (String[] rule : rules)
                            if (rule.length == 3 && table[i][k].contains(rule[1]) && table[k + 1][j].contains(rule[2]))
                                table[i][j].add(rule[0]);
                }
            return table[0][n - 1].contains(startSymbol);
        }
    }

    static class ParseResult {
        boolean accepted;
        int errors;
        String errorType;
        String correctedInput;

        ParseResult(boolean accepted, int errors, String errorType, String correctedInput) {
            this.accepted = accepted;
            this.errors = errors;
            this.errorType = errorType;
            this.correctedInput = correctedInput;
        }
    }

    private Grammar grammar;
    private int maxErrors;

    public ErrorTolerantParser(Grammar grammar, int maxErrors) {
        this.grammar = grammar;
        this.maxErrors = maxErrors;
    }

    public ParseResult parse(String[] input) {
        if (grammar.exactParse(input))
            return new ParseResult(true, 0, "精确匹配", Arrays.toString(input));

        if (maxErrors >= 1) {
            for (int i = 0; i < input.length; i++) {
                String[] deleted = new String[input.length - 1];
                for (int j = 0, k = 0; j < input.length; j++)
                    if (j != i) deleted[k++] = input[j];
                if (grammar.exactParse(deleted))
                    return new ParseResult(true, 1, "删除位置" + i + "的'" + input[i] + "'",
                        Arrays.toString(deleted));
            }

            for (int i = 0; i < input.length; i++) {
                for (String terminal : grammar.terminals) {
                    if (!terminal.equals(input[i])) {
                        String[] replaced = input.clone();
                        replaced[i] = terminal;
                        if (grammar.exactParse(replaced))
                            return new ParseResult(true, 1,
                                "替换位置" + i + "的'" + input[i] + "'为'" + terminal + "'",
                                Arrays.toString(replaced));
                    }
                }
            }

            for (int i = 0; i <= input.length; i++) {
                for (String terminal : grammar.terminals) {
                    String[] inserted = new String[input.length + 1];
                    for (int j = 0; j < i; j++) inserted[j] = input[j];
                    inserted[i] = terminal;
                    for (int j = i; j < input.length; j++) inserted[j + 1] = input[j];
                    if (grammar.exactParse(inserted))
                        return new ParseResult(true, 1,
                            "在位置" + i + "插入'" + terminal + "'",
                            Arrays.toString(inserted));
                }
            }
        }

        return new ParseResult(false, -1, "无法在" + maxErrors + "个错误内修正", "");
    }

    public static void main(String[] args) {
        System.out.println("=== 容错句法解析 ===\n");

        Grammar triangleG = new Grammar("三角形", "S");
        triangleG.addRule("S", "D", "R");
        triangleG.addRule("R", "D", "D");
        triangleG.addRule("D", "d");

        Grammar rectG = new Grammar("矩形", "S");
        rectG.addRule("S", "H", "T");
        rectG.addRule("T", "V", "U");
        rectG.addRule("U", "H", "V");
        rectG.addRule("H", "h");
        rectG.addRule("V", "v");

        Grammar zG = new Grammar("Z字形", "S");
        zG.addRule("S", "H", "R");
        zG.addRule("R", "D", "H");
        zG.addRule("H", "h");
        zG.addRule("D", "d");

        Map<String, Grammar> grammars = new LinkedHashMap<>();
        grammars.put("三角形", triangleG);
        grammars.put("矩形", rectG);
        grammars.put("Z字形", zG);

        String[][] tests = {
            {"d", "d", "d"},         // 三角形（精确）
            {"d", "d"},             // 三角形缺一笔
            {"d", "h", "d", "d"},   // 三角形多一笔
            {"d", "d", "h"},        // 三角形一笔写错
            {"h", "v", "h", "v"},   // 矩形（精确）
            {"h", "v", "v"},        // 矩形缺一笔
            {"h", "d", "h", "v"},   // 矩形一笔写错
            {"h", "d", "h"},        // Z字形（精确）
            {"h", "h"},             // Z字形缺一笔
            {"v", "v", "v"},        // 无法识别
        };

        for (String[] test : tests) {
            System.out.println("输入: " + Arrays.toString(test));
            boolean recognized = false;
            for (Map.Entry<String, Grammar> entry : grammars.entrySet()) {
                ErrorTolerantParser parser = new ErrorTolerantParser(entry.getValue(), 1);
                ParseResult result = parser.parse(test);
                if (result.accepted) {
                    System.out.println("  -> 识别为: " + entry.getKey());
                    System.out.println("     错误数: " + result.errors + ", 修正: " + result.errorType);
                    if (result.errors > 0)
                        System.out.println("     修正后: " + result.correctedInput);
                    recognized = true;
                    break;
                }
            }
            if (!recognized)
                System.out.println("  -> 无法识别（在允许1个错误的范围内无匹配文法）");
            System.out.println();
        }

        System.out.println("容错解析的意义:");
        System.out.println("  手写识别中，笔画提取不可避免地会出现错误（漏检、误检、误分类）。");
        System.out.println("  容错解析器通过允许有限数量的符号级错误，显著提高了识别系统的鲁棒性。");
        System.out.println("  错误数量可作为匹配置信度的度量——精确匹配(0错误)优于近似匹配(1错误)。");
    }
}

## 总结

句法模式识别是一种独特的模式识别方法论，其核心特点：

1. **结构化描述**：用符号和文法规则描述模式的结构，而非数值特征向量
2. **强表达能力**：能描述具有递归、嵌套等复杂结构的模式
3. **可解释性强**：文法规则直观反映了模式的构成逻辑
4. **分类即解析**：模式分类等价于语法分析问题

其主要局限性：
- **基元提取困难**：如何从原始数据中可靠地提取基元是关键难点
- **文法设计依赖专家知识**：需要领域专家手工设计文法规则
- **对噪声敏感**：基元提取的错误会直接导致解析失败
- **计算复杂度**：CYK算法为O(n³)，对长序列不够高效

## 解析算法对比

| 特性 | CYK算法 | Earley算法 | 递归下降 |
|------|---------|-----------|---------|
| **文法要求** | 必须为Chomsky范式(CNF) | 任意CFG | 任意CFG（需避免左递归） |
| **时间复杂度** | O(n³·\|G\|) | O(n³·\|G\|)，无歧义时O(n²) | 最坏O(2^n)，实际通常O(n) |
| **空间复杂度** | O(n²·\|G\|) | O(n²·\|G\|) | O(n)（栈深度） |
| **歧义文法** | 能处理 | 能处理，天然支持所有解析树 | 只返回第一个匹配 |
| **空产生式(ε)** | 需特殊处理 | 天然支持 | 天然支持 |
| **增量解析** | 不支持 | 支持（逐符号处理） | 支持 |
| **实现难度** | 中等（动态规划） | 较高（预测+扫描+完成） | 简单（直观递归） |
| **实际性能** | 稳定O(n³) | 对非歧义文法可达O(n) | 对LL(1)文法最快 |
| **错误恢复** | 需额外实现 | 可自然扩展 | 容易实现 |
| **适用场景** | 需要所有解析结果 | 通用解析、NLP | 编译器前端、简单模式 |

**选择建议**：
- 文法简单、无左递归 → 递归下降（实现简单、性能最优）
- 文法复杂、可能歧义 → Earley（最通用、灵活性最高）
- 文法已为CNF、需要完整解析矩阵 → CYK（理论简洁、教学用途）
- 需要容错解析 → 在任何算法基础上扩展编辑距离
- NLP领域 → Earley（天然处理歧义和ε规则）

句法模式识别与统计模式识别并非互斥，现代方法常将两者结合：用统计方法提取可靠的基元，再用句法方法进行结构化分类。在自然语言处理、编译原理、生物序列分析等领域，句法方法仍然发挥着不可替代的作用。
