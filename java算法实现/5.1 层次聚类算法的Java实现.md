# 5.1 层次聚类算法的Java实现

> **第5章 统计聚类算法** | 层次聚类 · 动态聚类(K-Means)

## 背景

层次聚类（Hierarchical Clustering）是一种经典的无监督聚类方法，它通过建立一个聚类的层次结构（树状图/谱系图）来组织数据。与K-Means等划分式聚类不同，层次聚类不需要预先指定聚类个数，用户可以通过在不同层次切割树状图来获得不同粒度的聚类结果。

层次聚类有两种策略：

1. **凝聚式（自底向上，Agglomerative）**：初始时每个样本为一个单独的簇，然后逐步合并最相似的两个簇，直到所有样本归为一个簇
2. **分裂式（自顶向下，Divisive）**：初始时所有样本在一个簇中，然后逐步将簇分裂，直到每个样本单独成簇

簇间距离的计算方式（链接准则）：
- **单链接（Single Linkage）**：两簇间最近两点的距离
- **全链接（Complete Linkage）**：两簇间最远两点的距离
- **平均链接（Average Linkage）**：两簇间所有点对距离的平均值
- **Ward方法**：合并后簇内方差增量最小

## Ward方法的数学原理

Ward方法（Ward's Minimum Variance Method）由Joe H. Ward于1963年提出，其核心思想是在每一步合并中选择使**总的簇内方差增量最小**的两个簇进行合并。

具体地，对于两个簇A和B，Ward距离定义为合并后簇内离差平方和（ESS）的增量：

d(A,B) = (n_A · n_B) / (n_A + n_B) · ||c_A - c_B||²

其中n_A和n_B分别是两个簇的样本数，c_A和c_B是两个簇的质心。这个公式表明：Ward距离同时考虑了簇间质心的距离和簇的大小，倾向于合并样本数较少且质心接近的簇，因此能产生大小相对均匀的簇。

Ward方法的递推公式（Lance-Williams公式）：当簇A和B合并为簇(AB)后，(AB)与另一个簇C之间的Ward距离为：

d((AB),C) = [(n_A+n_C)·d(A,C) + (n_B+n_C)·d(B,C) - n_C·d(A,B)] / (n_A+n_B+n_C)

Ward方法等价于在每步贪心地最小化全局的簇内离差平方和，是实践中使用最广泛的链接准则之一，尤其适合发现球形、大小相近的簇。

## Cophenetic相关系数

Cophenetic（共表征）相关系数是评估层次聚类质量的重要指标，由Sokal和Rohlf于1962年提出。它衡量的是原始数据的距离矩阵与层次聚类产生的Cophenetic距离矩阵之间的相关程度。

Cophenetic距离定义为：对于任意两个样本i和j，它们在树状图上首次被合并到同一个簇时的合并距离。如果原始距离小的样本对在树状图上也在较低层次（较小距离）合并，则说明聚类结果很好地保留了原始数据的距离结构。

Cophenetic相关系数的取值范围为[-1, 1]，越接近1表示聚类结果越忠实地反映了原始数据的距离关系。通常认为：
- 大于0.75：聚类质量优秀
- 0.6-0.75：聚类质量良好
- 小于0.6：聚类结构可能不太可靠

不同链接准则产生的Cophenetic相关系数差异较大，平均链接通常能获得最高的Cophenetic相关系数。

## 树状图（Dendrogram）解读

树状图是层次聚类最重要的可视化工具，它以树形结构直观展示聚类的合并过程：

1. **叶节点**：位于底部，代表每个原始样本
2. **内部节点**：代表一次簇合并操作，节点的高度对应合并时的距离
3. **切割线**：在某个高度画一条水平线，可以得到对应数量的聚类
4. **分支长度**：较长的分支表示合并的两个簇距离较远，通常是确定最优聚类数的线索——选择最长分支处进行切割

解读技巧：观察树状图中最大的"跳跃"（相邻合并距离的最大差异），在跳跃处切割往往能获得最自然的聚类划分。

层次聚类广泛应用于生物信息学（基因聚类）、社交网络分析、文档分类等领域。

## 层次聚类算法的java实现

下面实现凝聚式层次聚类，支持多种链接准则：

```
import java.util.*;

public class HierarchicalClustering {

    // 聚类合并记录
    static class MergeRecord {
        int cluster1, cluster2;
        double distance;
        int newCluster;

        MergeRecord(int c1, int c2, double dist, int nc) {
            this.cluster1 = c1; this.cluster2 = c2;
            this.distance = dist; this.newCluster = nc;
        }

        @Override
        public String toString() {
            return "合并 簇" + cluster1 + " 和 簇" + cluster2
                 + " (距离=" + String.format("%.4f", distance) + ") -> 簇" + newCluster;
        }
    }

    private double[][] data;
    private String linkage;
    private List<MergeRecord> mergeHistory = new ArrayList<>();

    public HierarchicalClustering(double[][] data, String linkage) {
        this.data = data;
        this.linkage = linkage;
    }

    // 欧氏距离
    private double distance(double[] a, double[] b) {
        double sum = 0;
        for (int i = 0; i < a.length; i++) sum += Math.pow(a[i] - b[i], 2);
        return Math.sqrt(sum);
    }

    // 计算两个簇之间的距离
    private double clusterDistance(List<Integer> cluster1, List<Integer> cluster2) {
        switch (linkage) {
            case "single": {
                double minDist = Double.MAX_VALUE;
                for (int i : cluster1)
                    for (int j : cluster2)
                        minDist = Math.min(minDist, distance(data[i], data[j]));
                return minDist;
            }
            case "complete": {
                double maxDist = 0;
                for (int i : cluster1)
                    for (int j : cluster2)
                        maxDist = Math.max(maxDist, distance(data[i], data[j]));
                return maxDist;
            }
            case "average": {
                double totalDist = 0;
                int count = 0;
                for (int i : cluster1)
                    for (int j : cluster2) {
                        totalDist += distance(data[i], data[j]);
                        count++;
                    }
                return totalDist / count;
            }
            default:
                throw new IllegalArgumentException("未知链接准则: " + linkage);
        }
    }

    // 执行凝聚式层次聚类
    public int[] fit(int numClusters) {
        int n = data.length;
        Map<Integer, List<Integer>> clusters = new LinkedHashMap<>();
        for (int i = 0; i < n; i++) {
            List<Integer> list = new ArrayList<>();
            list.add(i);
            clusters.put(i, list);
        }

        int nextClusterId = n;
        mergeHistory.clear();

        while (clusters.size() > numClusters) {
            // 找距离最小的两个簇
            double minDist = Double.MAX_VALUE;
            int mergeId1 = -1, mergeId2 = -1;

            List<Integer> clusterIds = new ArrayList<>(clusters.keySet());
            for (int i = 0; i < clusterIds.size(); i++) {
                for (int j = i + 1; j < clusterIds.size(); j++) {
                    double dist = clusterDistance(
                        clusters.get(clusterIds.get(i)),
                        clusters.get(clusterIds.get(j))
                    );
                    if (dist < minDist) {
                        minDist = dist;
                        mergeId1 = clusterIds.get(i);
                        mergeId2 = clusterIds.get(j);
                    }
                }
            }

            // 合并
            List<Integer> merged = new ArrayList<>(clusters.get(mergeId1));
            merged.addAll(clusters.get(mergeId2));
            mergeHistory.add(new MergeRecord(mergeId1, mergeId2, minDist, nextClusterId));

            clusters.remove(mergeId1);
            clusters.remove(mergeId2);
            clusters.put(nextClusterId, merged);
            nextClusterId++;
        }

        // 生成标签
        int[] labels = new int[n];
        int labelIdx = 0;
        for (Map.Entry<Integer, List<Integer>> entry : clusters.entrySet()) {
            for (int idx : entry.getValue()) labels[idx] = labelIdx;
            labelIdx++;
        }
        return labels;
    }

    public List<MergeRecord> getMergeHistory() { return mergeHistory; }

    public static void main(String[] args) {
        double[][] data = {
            {1.0, 1.0}, {1.5, 1.5}, {1.2, 1.3},
            {5.0, 5.0}, {5.5, 5.5}, {5.2, 5.3},
            {9.0, 1.0}, {9.5, 1.5}, {9.2, 1.3}
        };

        String[] linkages = {"single", "complete", "average"};

        for (String linkage : linkages) {
            System.out.println("=== 层次聚类 (" + linkage + " linkage) ===");
            HierarchicalClustering hc = new HierarchicalClustering(data, linkage);
            int[] labels = hc.fit(3);

            System.out.println("合并过程:");
            for (MergeRecord record : hc.getMergeHistory())
                System.out.println("  " + record);

            System.out.println("聚类结果 (3个簇):");
            for (int i = 0; i < data.length; i++)
                System.out.println("  样本" + i + " " + Arrays.toString(data[i]) + " -> 簇" + labels[i]);
            System.out.println();
        }

        // 不同簇数的聚类
        System.out.println("=== 不同聚类数目 (average linkage) ===");
        for (int k = 2; k <= 5; k++) {
            HierarchicalClustering hc = new HierarchicalClustering(data, "average");
            int[] labels = hc.fit(k);
            System.out.print("K=" + k + ": ");
            System.out.println(Arrays.toString(labels));
        }
    }
}
```

## 增强版：Ward链接、轮廓系数、Cophenetic相关系数、树状图可视化与基因表达聚类

在基础版本之上，增加Ward链接准则、轮廓系数（Silhouette Score）计算、Cophenetic相关系数、文本树状图可视化，以及一个实用的基因表达数据聚类示例：

```
import java.util.*;

public class EnhancedHierarchicalClustering {

    static class MergeRecord {
        int cluster1, cluster2;
        double distance;
        int newCluster;
        int size;

        MergeRecord(int c1, int c2, double dist, int nc, int size) {
            this.cluster1 = c1; this.cluster2 = c2;
            this.distance = dist; this.newCluster = nc; this.size = size;
        }
    }

    private double[][] data;
    private String linkage;
    private List<MergeRecord> mergeHistory = new ArrayList<>();

    public EnhancedHierarchicalClustering(double[][] data, String linkage) {
        this.data = data;
        this.linkage = linkage;
    }

    private double distance(double[] a, double[] b) {
        double sum = 0;
        for (int i = 0; i < a.length; i++) sum += Math.pow(a[i] - b[i], 2);
        return Math.sqrt(sum);
    }

    private double[] clusterCenter(List<Integer> cluster) {
        int dim = data[0].length;
        double[] center = new double[dim];
        for (int idx : cluster)
            for (int d = 0; d < dim; d++) center[d] += data[idx][d];
        for (int d = 0; d < dim; d++) center[d] /= cluster.size();
        return center;
    }

    private double clusterDistance(List<Integer> c1, List<Integer> c2) {
        switch (linkage) {
            case "single": {
                double min = Double.MAX_VALUE;
                for (int i : c1) for (int j : c2) min = Math.min(min, distance(data[i], data[j]));
                return min;
            }
            case "complete": {
                double max = 0;
                for (int i : c1) for (int j : c2) max = Math.max(max, distance(data[i], data[j]));
                return max;
            }
            case "average": {
                double total = 0; int count = 0;
                for (int i : c1) for (int j : c2) { total += distance(data[i], data[j]); count++; }
                return total / count;
            }
            case "ward": {
                double[] center1 = clusterCenter(c1);
                double[] center2 = clusterCenter(c2);
                double dist = 0;
                for (int d = 0; d < center1.length; d++)
                    dist += Math.pow(center1[d] - center2[d], 2);
                return Math.sqrt(2.0 * c1.size() * c2.size() / (c1.size() + c2.size()) * dist);
            }
            default:
                throw new IllegalArgumentException("未知链接准则: " + linkage);
        }
    }

    public int[] fit(int numClusters) {
        int n = data.length;
        Map<Integer, List<Integer>> clusters = new LinkedHashMap<>();
        for (int i = 0; i < n; i++) {
            List<Integer> list = new ArrayList<>();
            list.add(i);
            clusters.put(i, list);
        }

        int nextId = n;
        mergeHistory.clear();

        while (clusters.size() > numClusters) {
            double minDist = Double.MAX_VALUE;
            int id1 = -1, id2 = -1;
            List<Integer> keys = new ArrayList<>(clusters.keySet());
            for (int i = 0; i < keys.size(); i++)
                for (int j = i + 1; j < keys.size(); j++) {
                    double d = clusterDistance(clusters.get(keys.get(i)), clusters.get(keys.get(j)));
                    if (d < minDist) { minDist = d; id1 = keys.get(i); id2 = keys.get(j); }
                }

            List<Integer> merged = new ArrayList<>(clusters.get(id1));
            merged.addAll(clusters.get(id2));
            mergeHistory.add(new MergeRecord(id1, id2, minDist, nextId, merged.size()));
            clusters.remove(id1);
            clusters.remove(id2);
            clusters.put(nextId++, merged);
        }

        int[] labels = new int[n];
        int labelIdx = 0;
        for (Map.Entry<Integer, List<Integer>> entry : clusters.entrySet()) {
            for (int idx : entry.getValue()) labels[idx] = labelIdx;
            labelIdx++;
        }
        return labels;
    }

    public List<MergeRecord> getMergeHistory() { return mergeHistory; }

    public static double silhouetteScore(double[][] data, int[] labels) {
        int n = data.length;
        double totalScore = 0;

        for (int i = 0; i < n; i++) {
            double a = 0;
            int sameCount = 0;
            Map<Integer, double[]> otherClusterDists = new HashMap<>();
            Map<Integer, int[]> otherClusterCounts = new HashMap<>();

            for (int j = 0; j < n; j++) {
                if (i == j) continue;
                double dist = 0;
                for (int d = 0; d < data[0].length; d++)
                    dist += Math.pow(data[i][d] - data[j][d], 2);
                dist = Math.sqrt(dist);

                if (labels[j] == labels[i]) {
                    a += dist;
                    sameCount++;
                } else {
                    otherClusterDists.computeIfAbsent(labels[j], k -> new double[1])[0] += dist;
                    otherClusterCounts.computeIfAbsent(labels[j], k -> new int[1])[0]++;
                }
            }

            if (sameCount > 0) a /= sameCount;

            double b = Double.MAX_VALUE;
            for (Map.Entry<Integer, double[]> entry : otherClusterDists.entrySet()) {
                int lbl = entry.getKey();
                double avgDist = entry.getValue()[0] / otherClusterCounts.get(lbl)[0];
                b = Math.min(b, avgDist);
            }
            if (b == Double.MAX_VALUE) b = 0;

            double s = (sameCount == 0) ? 0 : (b - a) / Math.max(a, b);
            totalScore += s;
        }
        return totalScore / n;
    }

    public double copheneticCorrelation() {
        int n = data.length;
        int pairCount = n * (n - 1) / 2;
        double[] origDists = new double[pairCount];
        double[] cophDists = new double[pairCount];

        Map<Integer, Set<Integer>> clusterMembers = new HashMap<>();
        for (int i = 0; i < n; i++) {
            Set<Integer> s = new HashSet<>(); s.add(i);
            clusterMembers.put(i, s);
        }

        double[][] cophMatrix = new double[n][n];
        for (MergeRecord mr : mergeHistory) {
            Set<Integer> s1 = clusterMembers.get(mr.cluster1);
            Set<Integer> s2 = clusterMembers.get(mr.cluster2);
            for (int i : s1)
                for (int j : s2) {
                    cophMatrix[i][j] = mr.distance;
                    cophMatrix[j][i] = mr.distance;
                }
            Set<Integer> merged = new HashSet<>(s1);
            merged.addAll(s2);
            clusterMembers.put(mr.newCluster, merged);
        }

        int idx = 0;
        for (int i = 0; i < n; i++)
            for (int j = i + 1; j < n; j++) {
                origDists[idx] = distance(data[i], data[j]);
                cophDists[idx] = cophMatrix[i][j];
                idx++;
            }

        double meanOrig = 0, meanCoph = 0;
        for (int i = 0; i < pairCount; i++) { meanOrig += origDists[i]; meanCoph += cophDists[i]; }
        meanOrig /= pairCount; meanCoph /= pairCount;

        double num = 0, den1 = 0, den2 = 0;
        for (int i = 0; i < pairCount; i++) {
            double dO = origDists[i] - meanOrig, dC = cophDists[i] - meanCoph;
            num += dO * dC; den1 += dO * dO; den2 += dC * dC;
        }
        return num / Math.sqrt(den1 * den2);
    }

    public void printDendrogram() {
        if (mergeHistory.isEmpty()) return;
        System.out.println("树状图（Dendrogram）：");
        System.out.println("合并距离  |  合并过程");
        System.out.println("----------|-------------------------------------------");

        int n = data.length;
        Map<Integer, String> labels = new HashMap<>();
        for (int i = 0; i < n; i++) labels.put(i, "S" + i);

        double maxDist = mergeHistory.get(mergeHistory.size() - 1).distance;

        for (MergeRecord mr : mergeHistory) {
            int barLen = (maxDist > 0) ? (int)(mr.distance / maxDist * 30) : 1;
            barLen = Math.max(barLen, 1);
            StringBuilder bar = new StringBuilder();
            for (int i = 0; i < barLen; i++) bar.append("█");

            String l1 = labels.getOrDefault(mr.cluster1, "C" + mr.cluster1);
            String l2 = labels.getOrDefault(mr.cluster2, "C" + mr.cluster2);
            String merged = "{" + l1 + "," + l2 + "}";
            labels.put(mr.newCluster, merged);

            System.out.printf("%9.4f | %s %s%n", mr.distance, bar, merged);
        }
    }

        System.out.println("\n不同聚类数的轮廓系数：");
        for (int k = 2; k <= 6; k++) {
            EnhancedHierarchicalClustering tmp = new EnhancedHierarchicalClustering(geneExpression, "ward");
            int[] lbl = tmp.fit(k);
            System.out.printf("  K=%d: 轮廓系数=%.4f%n", k, silhouetteScore(geneExpression, lbl));
        }
    }
}
```

## 总结

层次聚类算法的主要优点：

1. **无需预先指定聚类数**：可以根据树状图在任意层次获取聚类结果
2. **结果可解释性强**：树状图直观展示了数据的层次结构关系
3. **可发现任意形状的簇**：尤其是单链接法

其主要局限性：
- **时间复杂度高**：基本实现为 O(n³)，不适合大规模数据集
- **不可逆性**：一旦合并或分裂，无法撤回
- **对噪声敏感**：特别是单链接法容易产生"链式效应"

实际应用中，可以通过先使用K-Means进行粗聚类，再对粗聚类结果进行层次聚类的方式来处理大规模数据集。

## 链接准则对比

| 链接准则 | 簇间距离定义 | 优点 | 缺点 | 适用场景 |
|---------|------------|------|------|---------|
| **单链接** | 两簇最近点距离 | 能发现细长的、非凸形状的簇 | 容易产生链式效应，对噪声敏感 | 检测细长或不规则形状的簇结构 |
| **全链接** | 两簇最远点距离 | 产生紧凑的球形簇，对噪声鲁棒 | 倾向于将大簇打碎 | 数据分布较均匀、需要紧凑簇时 |
| **平均链接** | 所有点对距离的均值 | 折中方案，Cophenetic相关系数通常最高 | 对簇的形状和大小无特别偏好 | 通用场景，尤其是不确定数据结构时 |
| **Ward方法** | 簇内方差增量 | 产生大小均匀的球形簇，效果类似K-Means | 只适合球形簇，计算开销稍大 | 已知数据分布为球形簇时首选 |

## 链接准则选择指南

1. **先验知识**：如果已知簇的形状，据此选择——球形簇用Ward或全链接，不规则形状用单链接
2. **Cophenetic相关系数**：分别用不同链接准则运行，选择Cophenetic系数最高的
3. **轮廓系数辅助**：在确定聚类数后，比较不同链接下的轮廓系数
4. **保守策略**：在缺乏先验知识时，平均链接是最安全的默认选择；Ward方法在大多数规则分布的数据上表现最优
5. **数据规模**：超过数千个样本时，建议先用K-Means粗聚类后再用层次聚类精化，或直接使用BIRCH等可扩展算法
