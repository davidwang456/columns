# 6.1 特征降维PCA算法的Java实现

> **第6章 统计模式识别** | 特征降维(PCA) · 模糊模式识别(FCM)

## 背景

主成分分析（Principal Component Analysis，PCA）是最经典的特征降维方法，由Karl Pearson在1901年提出。PCA通过线性变换将高维数据投影到低维空间，同时最大限度地保留数据的方差信息（即信息量）。

PCA的核心思想：
1. 找到数据方差最大的方向，作为第一主成分
2. 在与第一主成分正交的方向中，找方差最大的方向，作为第二主成分
3. 依此类推，直到获得所需数量的主成分

PCA的数学步骤：
1. **数据中心化**：将每个特征减去其均值
2. **计算协方差矩阵**：C = (1/n) X^T X
3. **特征值分解**：对协方差矩阵求特征值和特征向量
4. **选择主成分**：按特征值从大到小排序，选取前k个特征向量
5. **投影降维**：Y = X * W，其中W为选取的特征向量组成的矩阵

## 基于SVD的PCA理论

除了通过协方差矩阵的特征值分解实现PCA外，奇异值分解（SVD）提供了另一种更数值稳定的途径。对中心化后的数据矩阵 X（n×d），其SVD分解为：

X = U Σ V^T

其中 U（n×n）和 V（d×d）是正交矩阵，Σ（n×d）是对角矩阵。V 的列向量就是主成分方向（即协方差矩阵的特征向量），而奇异值 σ_i 与特征值的关系为 λ_i = σ_i² / (n-1)。

SVD方法的优势在于：
- **数值稳定性更好**：直接对数据矩阵操作，避免了计算协方差矩阵时可能的精度损失
- **适合高维数据**：当特征维度远大于样本数时（d >> n），SVD更高效
- **截断SVD**：可以只计算前k个奇异值/向量，避免完整分解的开销

## 核PCA（Kernel PCA）

标准PCA只能捕获数据中的线性结构。核PCA通过核技巧将数据隐式映射到高维特征空间，在该空间中执行线性PCA，从而实现非线性降维。

核PCA的步骤：
1. 计算核矩阵 K，其中 K_{ij} = κ(x_i, x_j)
2. 中心化核矩阵：K' = K - 1_n K - K 1_n + 1_n K 1_n
3. 对 K' 进行特征值分解
4. 选取前k个特征向量作为投影方向

常用核函数包括：
- **高斯核（RBF）**：κ(x,y) = exp(-||x-y||² / 2σ²)，能处理复杂的非线性结构
- **多项式核**：κ(x,y) = (x^T y + c)^d，适合多项式可分的数据
- **Sigmoid核**：κ(x,y) = tanh(αx^T y + c)，类似神经网络的激活

## 数据标准化的重要性

PCA对数据的尺度非常敏感。如果不同特征的量纲或数值范围差异很大（如身高以厘米计 vs 体重以千克计），方差大的特征会主导主成分方向，导致结果偏颇。

Z-score标准化将每个特征变换为均值为0、标准差为1：

z = (x - μ) / σ

标准化后，PCA在相关系数矩阵上执行（而非协方差矩阵），确保每个特征对分析的贡献权重相等。在实践中，除非所有特征已经在相同的尺度上，否则应始终先进行标准化。

## 白化变换（Whitening Transform）

白化变换是PCA的延伸，不仅旋转坐标轴使特征正交，还将每个主成分的方差归一化为1。白化后的数据满足：
- 特征之间不相关（协方差为0）
- 每个特征的方差为1

白化变换公式：Y_white = Y / sqrt(λ_i)，其中 Y 是PCA投影后的数据，λ_i 是对应的特征值。

白化在深度学习和信号处理中有重要应用，是独立成分分析（ICA）的常见预处理步骤。

PCA广泛应用于：
- 数据可视化（高维数据降至2D/3D）
- 噪声过滤
- 特征压缩
- 人脸识别（特征脸方法）

## 特征降维PCA算法的java实现

下面纯Java实现PCA算法，包括协方差矩阵计算、特征值分解（Jacobi迭代法）和降维投影：

```
import java.util.Arrays;

public class PCA {

    private double[] mean;
    private double[][] eigenvectors;
    private double[] eigenvalues;
    private int numComponents;

    public PCA(int numComponents) {
        this.numComponents = numComponents;
    }

    // 数据中心化
    private double[][] centerData(double[][] data) {
        int n = data.length, d = data[0].length;
        mean = new double[d];
        for (double[] row : data)
            for (int j = 0; j < d; j++) mean[j] += row[j];
        for (int j = 0; j < d; j++) mean[j] /= n;

        double[][] centered = new double[n][d];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < d; j++)
                centered[i][j] = data[i][j] - mean[j];
        return centered;
    }

    // 计算协方差矩阵
    private double[][] covarianceMatrix(double[][] centered) {
        int n = centered.length, d = centered[0].length;
        double[][] cov = new double[d][d];
        for (int i = 0; i < d; i++)
            for (int j = 0; j < d; j++) {
                double sum = 0;
                for (int k = 0; k < n; k++)
                    sum += centered[k][i] * centered[k][j];
                cov[i][j] = sum / (n - 1);
            }
        return cov;
    }

    /**
     * Jacobi迭代法求对称矩阵的特征值和特征向量
     * 通过一系列旋转变换使矩阵对角化
     */
    private void jacobiEigen(double[][] matrix) {
        int n = matrix.length;
        double[][] a = new double[n][n];
        for (int i = 0; i < n; i++) a[i] = matrix[i].clone();

        double[][] v = new double[n][n];
        for (int i = 0; i < n; i++) v[i][i] = 1.0;

        int maxIter = 100 * n * n;
        for (int iter = 0; iter < maxIter; iter++) {
            // 找最大非对角元素
            int p = 0, q = 1;
            double maxVal = Math.abs(a[0][1]);
            for (int i = 0; i < n; i++)
                for (int j = i + 1; j < n; j++)
                    if (Math.abs(a[i][j]) > maxVal) {
                        maxVal = Math.abs(a[i][j]);
                        p = i; q = j;
                    }

            if (maxVal < 1e-12) break;

            // 计算旋转角度
            double theta;
            if (Math.abs(a[p][p] - a[q][q]) < 1e-15)
                theta = Math.PI / 4;
            else
                theta = 0.5 * Math.atan2(2 * a[p][q], a[p][p] - a[q][q]);

            double cosT = Math.cos(theta), sinT = Math.sin(theta);

            // 执行旋转
            double[][] newA = new double[n][n];
            for (int i = 0; i < n; i++) newA[i] = a[i].clone();

            for (int i = 0; i < n; i++) {
                if (i != p && i != q) {
                    newA[i][p] = newA[p][i] = cosT * a[i][p] + sinT * a[i][q];
                    newA[i][q] = newA[q][i] = -sinT * a[i][p] + cosT * a[i][q];
                }
            }
            newA[p][p] = cosT * cosT * a[p][p] + 2 * sinT * cosT * a[p][q] + sinT * sinT * a[q][q];
            newA[q][q] = sinT * sinT * a[p][p] - 2 * sinT * cosT * a[p][q] + cosT * cosT * a[q][q];
            newA[p][q] = newA[q][p] = 0;
            a = newA;

            // 更新特征向量
            for (int i = 0; i < n; i++) {
                double vip = v[i][p], viq = v[i][q];
                v[i][p] = cosT * vip + sinT * viq;
                v[i][q] = -sinT * vip + cosT * viq;
            }
        }

        eigenvalues = new double[n];
        for (int i = 0; i < n; i++) eigenvalues[i] = a[i][i];

        // 按特征值从大到小排序
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;
        Arrays.sort(indices, (x, y) -> Double.compare(eigenvalues[y], eigenvalues[x]));

        double[] sortedEigenvalues = new double[n];
        eigenvectors = new double[n][n];
        for (int i = 0; i < n; i++) {
            sortedEigenvalues[i] = eigenvalues[indices[i]];
            for (int j = 0; j < n; j++)
                eigenvectors[j][i] = v[j][indices[i]];
        }
        eigenvalues = sortedEigenvalues;
    }

    // 拟合PCA模型
    public void fit(double[][] data) {
        double[][] centered = centerData(data);
        double[][] cov = covarianceMatrix(centered);
        jacobiEigen(cov);
    }

    // 降维变换
    public double[][] transform(double[][] data) {
        int n = data.length, d = data[0].length;
        double[][] result = new double[n][numComponents];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < numComponents; j++)
                for (int k = 0; k < d; k++)
                    result[i][j] += (data[i][k] - mean[k]) * eigenvectors[k][j];
        return result;
    }

    // 计算各主成分的方差贡献率
    public double[] explainedVarianceRatio() {
        double total = 0;
        for (double ev : eigenvalues) total += ev;
        double[] ratio = new double[numComponents];
        for (int i = 0; i < numComponents; i++) ratio[i] = eigenvalues[i] / total;
        return ratio;
    }

    // 累积方差贡献率
    public double[] cumulativeVarianceRatio() {
        double[] ratio = explainedVarianceRatio();
        double[] cumulative = new double[numComponents];
        cumulative[0] = ratio[0];
        for (int i = 1; i < numComponents; i++) cumulative[i] = cumulative[i - 1] + ratio[i];
        return cumulative;
    }

    public static void main(String[] args) {
        // 4维数据（模拟鸢尾花数据）
        double[][] data = {
            {5.1, 3.5, 1.4, 0.2}, {4.9, 3.0, 1.4, 0.2}, {4.7, 3.2, 1.3, 0.2},
            {4.6, 3.1, 1.5, 0.2}, {5.0, 3.6, 1.4, 0.2}, {5.4, 3.9, 1.7, 0.4},
            {7.0, 3.2, 4.7, 1.4}, {6.4, 3.2, 4.5, 1.5}, {6.9, 3.1, 4.9, 1.5},
            {5.5, 2.3, 4.0, 1.3}, {6.5, 2.8, 4.6, 1.5}, {5.7, 2.8, 4.5, 1.3},
            {6.3, 3.3, 6.0, 2.5}, {5.8, 2.7, 5.1, 1.9}, {7.1, 3.0, 5.9, 2.1},
            {6.5, 3.0, 5.8, 2.2}, {7.6, 3.0, 6.6, 2.1}, {7.2, 3.6, 6.1, 2.5}
        };

        System.out.println("=== PCA特征降维 ===");
        System.out.println("原始数据维度: " + data[0].length + "维, " + data.length + "个样本\n");

        PCA pca = new PCA(2);
        pca.fit(data);

        System.out.println("特征值:");
        for (int i = 0; i < pca.eigenvalues.length; i++)
            System.out.println("  λ" + (i + 1) + " = " + String.format("%.4f", pca.eigenvalues[i]));

        System.out.println("\n方差贡献率:");
        double[] ratio = pca.explainedVarianceRatio();
        double[] cumRatio = pca.cumulativeVarianceRatio();
        for (int i = 0; i < ratio.length; i++)
            System.out.println("  PC" + (i + 1) + ": " + String.format("%.2f%%", ratio[i] * 100)
                + " (累积: " + String.format("%.2f%%", cumRatio[i] * 100) + ")");

        System.out.println("\n降维后数据 (4D -> 2D):");
        double[][] transformed = pca.transform(data);
        for (int i = 0; i < transformed.length; i++)
            System.out.println("  样本" + String.format("%2d", i) + ": ["
                + String.format("%7.4f", transformed[i][0]) + ", "
                + String.format("%7.4f", transformed[i][1]) + "]");
    }
}

下面实现带Z-score标准化的PCA，先将数据标准化到统一尺度再执行降维，同时支持数据重建和重建误差计算：

```
import java.util.Arrays;

public class StandardizedPCA {

    private double[] mean;
    private double[] std;
    private double[][] eigenvectors;
    private double[] eigenvalues;
    private int numComponents;

    public StandardizedPCA(int numComponents) {
        this.numComponents = numComponents;
    }

    private double[][] zScoreStandardize(double[][] data) {
        int n = data.length, d = data[0].length;
        mean = new double[d];
        std = new double[d];

        for (double[] row : data)
            for (int j = 0; j < d; j++) mean[j] += row[j];
        for (int j = 0; j < d; j++) mean[j] /= n;

        for (double[] row : data)
            for (int j = 0; j < d; j++) std[j] += Math.pow(row[j] - mean[j], 2);
        for (int j = 0; j < d; j++) std[j] = Math.sqrt(std[j] / (n - 1));

        double[][] standardized = new double[n][d];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < d; j++)
                standardized[i][j] = (std[j] > 1e-10) ? (data[i][j] - mean[j]) / std[j] : 0;
        return standardized;
    }

    private double[][] covarianceMatrix(double[][] centered) {
        int n = centered.length, d = centered[0].length;
        double[][] cov = new double[d][d];
        for (int i = 0; i < d; i++)
            for (int j = 0; j < d; j++) {
                double sum = 0;
                for (int k = 0; k < n; k++) sum += centered[k][i] * centered[k][j];
                cov[i][j] = sum / (n - 1);
            }
        return cov;
    }

    private void jacobiEigen(double[][] matrix) {
        int n = matrix.length;
        double[][] a = new double[n][n];
        for (int i = 0; i < n; i++) a[i] = matrix[i].clone();
        double[][] v = new double[n][n];
        for (int i = 0; i < n; i++) v[i][i] = 1.0;

        for (int iter = 0; iter < 100 * n * n; iter++) {
            int p = 0, q = 1;
            double maxVal = Math.abs(a[0][1]);
            for (int i = 0; i < n; i++)
                for (int j = i + 1; j < n; j++)
                    if (Math.abs(a[i][j]) > maxVal) { maxVal = Math.abs(a[i][j]); p = i; q = j; }
            if (maxVal < 1e-12) break;

            double theta = (Math.abs(a[p][p] - a[q][q]) < 1e-15) ?
                Math.PI / 4 : 0.5 * Math.atan2(2 * a[p][q], a[p][p] - a[q][q]);
            double cosT = Math.cos(theta), sinT = Math.sin(theta);

            double[][] newA = new double[n][n];
            for (int i = 0; i < n; i++) newA[i] = a[i].clone();
            for (int i = 0; i < n; i++) {
                if (i != p && i != q) {
                    newA[i][p] = newA[p][i] = cosT * a[i][p] + sinT * a[i][q];
                    newA[i][q] = newA[q][i] = -sinT * a[i][p] + cosT * a[i][q];
                }
            }
            newA[p][p] = cosT * cosT * a[p][p] + 2 * sinT * cosT * a[p][q] + sinT * sinT * a[q][q];
            newA[q][q] = sinT * sinT * a[p][p] - 2 * sinT * cosT * a[p][q] + cosT * cosT * a[q][q];
            newA[p][q] = newA[q][p] = 0;
            a = newA;
            for (int i = 0; i < n; i++) {
                double vip = v[i][p], viq = v[i][q];
                v[i][p] = cosT * vip + sinT * viq;
                v[i][q] = -sinT * vip + cosT * viq;
            }
        }

        eigenvalues = new double[n];
        for (int i = 0; i < n; i++) eigenvalues[i] = a[i][i];
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;
        Arrays.sort(indices, (x, y) -> Double.compare(eigenvalues[y], eigenvalues[x]));
        double[] sorted = new double[n];
        eigenvectors = new double[n][n];
        for (int i = 0; i < n; i++) {
            sorted[i] = eigenvalues[indices[i]];
            for (int j = 0; j < n; j++) eigenvectors[j][i] = v[j][indices[i]];
        }
        eigenvalues = sorted;
    }

    public void fit(double[][] data) {
        double[][] standardized = zScoreStandardize(data);
        double[][] cov = covarianceMatrix(standardized);
        jacobiEigen(cov);
    }

    public double[][] transform(double[][] data) {
        int n = data.length, d = data[0].length;
        double[][] result = new double[n][numComponents];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < numComponents; j++)
                for (int k = 0; k < d; k++) {
                    double standardizedVal = (std[k] > 1e-10) ? (data[i][k] - mean[k]) / std[k] : 0;
                    result[i][j] += standardizedVal * eigenvectors[k][j];
                }
        return result;
    }

    public double[][] reconstruct(double[][] projected) {
        int n = projected.length, d = mean.length;
        double[][] reconstructed = new double[n][d];
        for (int i = 0; i < n; i++) {
            for (int k = 0; k < d; k++) {
                double val = 0;
                for (int j = 0; j < numComponents; j++)
                    val += projected[i][j] * eigenvectors[k][j];
                reconstructed[i][k] = val * std[k] + mean[k];
            }
        }
        return reconstructed;
    }

    public double reconstructionError(double[][] original, double[][] reconstructed) {
        double totalError = 0;
        int n = original.length, d = original[0].length;
        for (int i = 0; i < n; i++)
            for (int j = 0; j < d; j++)
                totalError += Math.pow(original[i][j] - reconstructed[i][j], 2);
        return totalError / n;
    }

    public double[][] whiten(double[][] projected) {
        int n = projected.length;
        double[][] whitened = new double[n][numComponents];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < numComponents; j++)
                whitened[i][j] = projected[i][j] / Math.sqrt(eigenvalues[j] + 1e-10);
        return whitened;
    }

    public double[] getEigenvalues() { return eigenvalues; }

    public static void main(String[] args) {
        double[][] data = {
            {170, 65, 30}, {175, 70, 25}, {160, 55, 35}, {180, 80, 28},
            {165, 60, 32}, {172, 68, 27}, {168, 63, 31}, {178, 75, 26},
            {162, 58, 34}, {176, 72, 29}
        };

        System.out.println("=== 标准化PCA + 重建 + 白化 ===\n");
        System.out.println("原始数据（身高cm/体重kg/年龄year），尺度差异大：");
        for (int i = 0; i < data.length; i++)
            System.out.println("  样本" + i + ": " + Arrays.toString(data[i]));

        StandardizedPCA spca = new StandardizedPCA(2);
        spca.fit(data);

        System.out.println("\n特征值: " + Arrays.toString(
            Arrays.stream(spca.getEigenvalues()).mapToObj(v -> String.format("%.4f", v)).toArray()));

        double[][] projected = spca.transform(data);
        System.out.println("\nPCA投影 (3D -> 2D):");
        for (int i = 0; i < projected.length; i++)
            System.out.println("  样本" + i + ": [" + String.format("%.4f", projected[i][0])
                + ", " + String.format("%.4f", projected[i][1]) + "]");

        double[][] reconstructed = spca.reconstruct(projected);
        double error = spca.reconstructionError(data, reconstructed);
        System.out.println("\n重建数据:");
        for (int i = 0; i < reconstructed.length; i++)
            System.out.println("  样本" + i + ": [" + String.format("%.2f", reconstructed[i][0])
                + ", " + String.format("%.2f", reconstructed[i][1])
                + ", " + String.format("%.2f", reconstructed[i][2]) + "]");
        System.out.println("平均重建误差(MSE): " + String.format("%.6f", error));

        double[][] whitened = spca.whiten(projected);
        System.out.println("\n白化后数据:");
        for (int i = 0; i < whitened.length; i++)
            System.out.println("  样本" + i + ": [" + String.format("%.4f", whitened[i][0])
                + ", " + String.format("%.4f", whitened[i][1]) + "]");

        double[] vars = new double[2];
        for (int j = 0; j < 2; j++) {
            double m = 0;
            for (int i = 0; i < whitened.length; i++) m += whitened[i][j];
            m /= whitened.length;
            for (int i = 0; i < whitened.length; i++) vars[j] += Math.pow(whitened[i][j] - m, 2);
            vars[j] /= (whitened.length - 1);
        }
        System.out.println("白化后各维度方差: [" + String.format("%.4f", vars[0])
            + ", " + String.format("%.4f", vars[1]) + "] (应接近1.0)");
    }
}

下面实现自动选择主成分数量（保留95%方差）的PCA，无需手动指定降维目标维度：

```
import java.util.Arrays;

public class AutoPCA {

    private double[] mean;
    private double[] std;
    private double[][] eigenvectors;
    private double[] eigenvalues;
    private int selectedComponents;
    private double varianceThreshold;

    public AutoPCA(double varianceThreshold) {
        this.varianceThreshold = varianceThreshold;
    }

    private double[][] standardizeAndCenter(double[][] data) {
        int n = data.length, d = data[0].length;
        mean = new double[d];
        std = new double[d];
        for (double[] row : data)
            for (int j = 0; j < d; j++) mean[j] += row[j];
        for (int j = 0; j < d; j++) mean[j] /= n;
        for (double[] row : data)
            for (int j = 0; j < d; j++) std[j] += Math.pow(row[j] - mean[j], 2);
        for (int j = 0; j < d; j++) std[j] = Math.sqrt(std[j] / (n - 1));

        double[][] result = new double[n][d];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < d; j++)
                result[i][j] = (std[j] > 1e-10) ? (data[i][j] - mean[j]) / std[j] : 0;
        return result;
    }

    private void jacobiEigen(double[][] matrix) {
        int n = matrix.length;
        double[][] a = new double[n][n];
        for (int i = 0; i < n; i++) a[i] = matrix[i].clone();
        double[][] v = new double[n][n];
        for (int i = 0; i < n; i++) v[i][i] = 1.0;

        for (int iter = 0; iter < 100 * n * n; iter++) {
            int p = 0, q = 1;
            double maxVal = Math.abs(a[0][1]);
            for (int i = 0; i < n; i++)
                for (int j = i + 1; j < n; j++)
                    if (Math.abs(a[i][j]) > maxVal) { maxVal = Math.abs(a[i][j]); p = i; q = j; }
            if (maxVal < 1e-12) break;

            double theta = (Math.abs(a[p][p] - a[q][q]) < 1e-15) ?
                Math.PI / 4 : 0.5 * Math.atan2(2 * a[p][q], a[p][p] - a[q][q]);
            double cosT = Math.cos(theta), sinT = Math.sin(theta);
            double[][] newA = new double[n][n];
            for (int i = 0; i < n; i++) newA[i] = a[i].clone();
            for (int i = 0; i < n; i++) {
                if (i != p && i != q) {
                    newA[i][p] = newA[p][i] = cosT * a[i][p] + sinT * a[i][q];
                    newA[i][q] = newA[q][i] = -sinT * a[i][p] + cosT * a[i][q];
                }
            }
            newA[p][p] = cosT * cosT * a[p][p] + 2 * sinT * cosT * a[p][q] + sinT * sinT * a[q][q];
            newA[q][q] = sinT * sinT * a[p][p] - 2 * sinT * cosT * a[p][q] + cosT * cosT * a[q][q];
            newA[p][q] = newA[q][p] = 0;
            a = newA;
            for (int i = 0; i < n; i++) {
                double vip = v[i][p], viq = v[i][q];
                v[i][p] = cosT * vip + sinT * viq;
                v[i][q] = -sinT * vip + cosT * viq;
            }
        }

        eigenvalues = new double[n];
        for (int i = 0; i < n; i++) eigenvalues[i] = a[i][i];
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;
        Arrays.sort(indices, (x, y) -> Double.compare(eigenvalues[y], eigenvalues[x]));
        double[] sorted = new double[n];
        eigenvectors = new double[n][n];
        for (int i = 0; i < n; i++) {
            sorted[i] = eigenvalues[indices[i]];
            for (int j = 0; j < n; j++) eigenvectors[j][i] = v[j][indices[i]];
        }
        eigenvalues = sorted;
    }

    private int selectComponents() {
        double totalVariance = 0;
        for (double ev : eigenvalues) totalVariance += ev;
        double cumulative = 0;
        for (int i = 0; i < eigenvalues.length; i++) {
            cumulative += eigenvalues[i];
            if (cumulative / totalVariance >= varianceThreshold) return i + 1;
        }
        return eigenvalues.length;
    }

    public void fit(double[][] data) {
        double[][] standardized = standardizeAndCenter(data);
        int d = data[0].length;
        double[][] cov = new double[d][d];
        for (int i = 0; i < d; i++)
            for (int j = 0; j < d; j++) {
                double sum = 0;
                for (int k = 0; k < standardized.length; k++)
                    sum += standardized[k][i] * standardized[k][j];
                cov[i][j] = sum / (standardized.length - 1);
            }
        jacobiEigen(cov);
        selectedComponents = selectComponents();
    }

    public double[][] transform(double[][] data) {
        int n = data.length, d = data[0].length;
        double[][] result = new double[n][selectedComponents];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < selectedComponents; j++)
                for (int k = 0; k < d; k++) {
                    double sv = (std[k] > 1e-10) ? (data[i][k] - mean[k]) / std[k] : 0;
                    result[i][j] += sv * eigenvectors[k][j];
                }
        return result;
    }

    public int getSelectedComponents() { return selectedComponents; }
    public double[] getEigenvalues() { return eigenvalues; }

    public static void main(String[] args) {

        System.out.println("=== 自动选择主成分数量 (保留95%方差) ===\n");

        for (int i = 0; i < projected.length; i++) {

    }

下面展示PCA作为噪声过滤器的实际应用——在含噪信号上执行PCA降维再重建，利用丢弃小特征值分量来去除噪声：

```
import java.util.Arrays;
import java.util.Random;

public class PCANoiseFilter {

    private double[] mean;
    private double[][] eigenvectors;
    private double[] eigenvalues;

    private void jacobiEigen(double[][] matrix) {
        int n = matrix.length;
        double[][] a = new double[n][n];
        for (int i = 0; i < n; i++) a[i] = matrix[i].clone();
        double[][] v = new double[n][n];
        for (int i = 0; i < n; i++) v[i][i] = 1.0;

        for (int iter = 0; iter < 100 * n * n; iter++) {
            int p = 0, q = 1;
            double maxVal = Math.abs(a[0][1]);
            for (int i = 0; i < n; i++)
                for (int j = i + 1; j < n; j++)
                    if (Math.abs(a[i][j]) > maxVal) { maxVal = Math.abs(a[i][j]); p = i; q = j; }
            if (maxVal < 1e-12) break;

            double theta = (Math.abs(a[p][p] - a[q][q]) < 1e-15) ?
                Math.PI / 4 : 0.5 * Math.atan2(2 * a[p][q], a[p][p] - a[q][q]);
            double cosT = Math.cos(theta), sinT = Math.sin(theta);
            double[][] newA = new double[n][n];
            for (int i = 0; i < n; i++) newA[i] = a[i].clone();
            for (int i = 0; i < n; i++) {
                if (i != p && i != q) {
                    newA[i][p] = newA[p][i] = cosT * a[i][p] + sinT * a[i][q];
                    newA[i][q] = newA[q][i] = -sinT * a[i][p] + cosT * a[i][q];
                }
            }
            newA[p][p] = cosT * cosT * a[p][p] + 2 * sinT * cosT * a[p][q] + sinT * sinT * a[q][q];
            newA[q][q] = sinT * sinT * a[p][p] - 2 * sinT * cosT * a[p][q] + cosT * cosT * a[q][q];
            newA[p][q] = newA[q][p] = 0;
            a = newA;
            for (int i = 0; i < n; i++) {
                double vip = v[i][p], viq = v[i][q];
                v[i][p] = cosT * vip + sinT * viq;
                v[i][q] = -sinT * vip + cosT * viq;
            }
        }

        eigenvalues = new double[n];
        for (int i = 0; i < n; i++) eigenvalues[i] = a[i][i];
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;
        Arrays.sort(indices, (x, y) -> Double.compare(eigenvalues[y], eigenvalues[x]));
        double[] sorted = new double[n];
        eigenvectors = new double[n][n];
        for (int i = 0; i < n; i++) {
            sorted[i] = eigenvalues[indices[i]];
            for (int j = 0; j < n; j++) eigenvectors[j][i] = v[j][indices[i]];
        }
        eigenvalues = sorted;
    }

    public double[][] denoise(double[][] noisyData, int keepComponents) {
        int n = noisyData.length, d = noisyData[0].length;

        mean = new double[d];
        for (double[] row : noisyData)
            for (int j = 0; j < d; j++) mean[j] += row[j];
        for (int j = 0; j < d; j++) mean[j] /= n;

        double[][] centered = new double[n][d];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < d; j++) centered[i][j] = noisyData[i][j] - mean[j];

        double[][] cov = new double[d][d];
        for (int i = 0; i < d; i++)
            for (int j = 0; j < d; j++) {
                double sum = 0;
                for (int k = 0; k < n; k++) sum += centered[k][i] * centered[k][j];
                cov[i][j] = sum / (n - 1);
            }
        jacobiEigen(cov);

        double[][] projected = new double[n][keepComponents];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < keepComponents; j++)
                for (int k = 0; k < d; k++)
                    projected[i][j] += centered[i][k] * eigenvectors[k][j];

        double[][] reconstructed = new double[n][d];
        for (int i = 0; i < n; i++)
            for (int k = 0; k < d; k++) {
                double val = 0;
                for (int j = 0; j < keepComponents; j++)
                    val += projected[i][j] * eigenvectors[k][j];
                reconstructed[i][k] = val + mean[k];
            }
        return reconstructed;
    }

    private static double mse(double[][] a, double[][] b) {
        double sum = 0;
        int count = 0;
        for (int i = 0; i < a.length; i++)
            for (int j = 0; j < a[0].length; j++) {
                sum += Math.pow(a[i][j] - b[i][j], 2);
                count++;
            }
        return sum / count;
    }

    public static void main(String[] args) {
        Random rng = new Random(42);
        int n = 20, d = 5;

        double[][] clean = new double[n][d];
        for (int i = 0; i < n; i++) {
            double t = (double) i / n * 2 * Math.PI;
            clean[i][0] = 3 * Math.sin(t);
            clean[i][1] = 3 * Math.cos(t);
            clean[i][2] = 2 * Math.sin(t) + Math.cos(t);
            clean[i][3] = clean[i][0] + clean[i][1];
            clean[i][4] = clean[i][2] - clean[i][1];
        }

        double noiseLevel = 0.5;
        double[][] noisy = new double[n][d];
        for (int i = 0; i < n; i++)
            for (int j = 0; j < d; j++)
                noisy[i][j] = clean[i][j] + rng.nextGaussian() * noiseLevel;

        System.out.println("=== PCA作为噪声过滤器 ===\n");
        System.out.println("数据维度: " + d + ", 样本数: " + n);
        System.out.println("噪声标准差: " + noiseLevel);
        System.out.println("含噪数据与干净数据的MSE: " + String.format("%.6f", mse(clean, noisy)));

        PCANoiseFilter filter = new PCANoiseFilter();

        System.out.println("\n保留不同主成分数量的去噪效果:");
        System.out.println(String.format("%-12s %-15s %-15s", "保留分量数", "重建MSE", "信噪比改善"));
        System.out.println("-".repeat(42));

        double noisyMse = mse(clean, noisy);
        for (int k = 1; k <= d; k++) {
            double[][] denoised = filter.denoise(noisy, k);
            double denoisedMse = mse(clean, denoised);
            double improvement = (1.0 - denoisedMse / noisyMse) * 100;
            System.out.println(String.format("%-12d %-15.6f %-15s", k,
                denoisedMse, String.format("%.1f%%", improvement)));
        }

        System.out.println("\n特征值分布:");
        double totalVar = 0;
        for (double ev : filter.eigenvalues) totalVar += ev;
        for (int i = 0; i < filter.eigenvalues.length; i++)
            System.out.println("  λ" + (i + 1) + " = " + String.format("%.4f", filter.eigenvalues[i])
                + " (" + String.format("%.1f%%", filter.eigenvalues[i] / totalVar * 100) + ")");

        System.out.println("\n结论: 保留前2-3个主成分即可有效去噪，");
        System.out.println("因为信号集中在少数主成分上，而噪声分散在所有维度。");
    }
}

## 总结

PCA主成分分析是最基本、最重要的特征降维方法，其核心优势：

1. **有效降维**：去除冗余特征，减少计算量，同时保留最重要的信息
2. **去除相关性**：降维后的各主成分相互正交，消除了特征间的线性相关
3. **噪声抑制**：丢弃小特征值对应的分量，等效于去除噪声
4. **无监督方法**：不需要类别标签信息

PCA的局限性：
- 只能处理线性关系，对非线性结构无法有效降维（可使用核PCA扩展）
- 主成分的物理含义不明确，可解释性较差
- 对数据的尺度敏感，通常需要先进行标准化

## PCA变体对比

| 特性 | 标准PCA | 核PCA (Kernel PCA) | 增量PCA (Incremental PCA) |
|------|---------|---------------------|---------------------------|
| **适用数据** | 线性可分结构 | 非线性结构 | 大规模数据 / 流数据 |
| **核心原理** | 协方差矩阵特征值分解 | 核矩阵特征值分解 | 分批次增量更新 |
| **计算复杂度** | O(d³) 或 O(nd²) | O(n³)（n为样本数） | O(ndk)（k为分量数） |
| **内存需求** | 需存储协方差矩阵 d×d | 需存储核矩阵 n×n | 常数级内存（批处理） |
| **非线性能力** | 无 | 通过核函数隐式映射 | 无 |
| **逆变换** | 直接可逆 | 近似逆变换 | 直接可逆 |
| **参数选择** | 主成分数k | 核函数类型及参数, k | 批大小, k |
| **典型应用** | 通用降维、数据可视化 | 非线性流形数据降维 | 大规模数据在线降维 |
| **优势** | 简单高效、理论完善 | 能发现非线性模式 | 内存高效、支持在线学习 |
| **劣势** | 无法处理非线性 | 计算开销大、参数敏感 | 结果依赖批次顺序 |

**选择建议**：
- 数据量适中、特征间线性相关 → 标准PCA
- 数据呈现非线性流形结构（如瑞士卷、同心圆） → 核PCA
- 数据量极大或以流式到达 → 增量PCA
- 不确定时，先尝试标准PCA，若降维后分类效果不佳再考虑核PCA

在模式识别中，PCA常作为预处理步骤，用于在分类前降低特征维度，既加速后续分类器的训练，又通过去除噪声提高分类精度。
