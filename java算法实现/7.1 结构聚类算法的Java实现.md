# 7.1 结构聚类算法的Java实现

> **第7章 结构模式识别** | 结构聚类算法 · 句法模式识别

## 背景

结构聚类算法是结构模式识别的重要方法，它利用数据点之间的结构关系（如连接、距离拓扑）来发现数据的聚类结构。与基于距离度量的统计聚类方法不同，结构聚类更关注数据的拓扑结构和连接模式。

最小生成树（Minimum Spanning Tree，MST）聚类是最经典的结构聚类方法之一。其基本思想是：

1. 将所有数据点视为图中的节点
2. 构建完全图，边的权重为节点间的距离
3. 使用Prim或Kruskal算法计算最小生成树
4. 删除MST中权重最大的若干条边，将树分割为若干子树，每个子树对应一个聚类

MST聚类的优势在于能发现任意形状的簇，不受簇的几何形状限制。此外，基于图的结构聚类还包括DBSCAN（基于密度的方法）、谱聚类等。

## Prim算法理论

Prim算法是另一种经典的最小生成树构建方法，与Kruskal算法不同，它采用"逐步扩展"策略：

1. 从任意节点出发，将其加入MST集合
2. 在MST集合与非MST集合之间的所有边中，选择权重最小的边
3. 将该边连接的非MST节点加入MST集合
4. 重复步骤2-3，直到所有节点都加入MST

Prim算法的复杂度取决于实现方式：使用邻接矩阵为 O(V²)，使用优先队列（二叉堆）为 O(E log V)。对于稠密图（如聚类中的完全图），邻接矩阵版本更高效；对于稀疏图，优先队列版本更优。

Kruskal和Prim算法产生相同的MST（若边权唯一），但Prim在聚类场景中有额外优势——它自然地保留了树的生长顺序信息，有助于分析数据的层次结构。

## 基于密度的边删除策略

简单的"删除最长边"策略虽然直观，但在实际数据中往往不够鲁棒。借鉴DBSCAN的密度思想，可以设计更智能的边删除策略：

**局部邻域统计法**：不再使用全局阈值，而是对MST中的每条边，考察其两端节点在MST上的局部邻域。如果一条边的权重显著大于其邻域内其他边的平均权重，则认为这条边连接了两个不同的簇。

具体而言，对于MST中的边 (u, v)，考察 u 和 v 在MST上深度为 d 的邻域内所有边的权重均值 μ 和标准差 σ。如果 w(u,v) > μ + k·σ（k 为敏感度参数），则标记为不一致边。这种方法能自适应地适应不同密度的簇。

## 与DBSCAN的对比

MST聚类和DBSCAN都能发现任意形状的簇，但原理不同：

- **MST聚类**：先建立全局连接结构（MST），再通过删边分割。属于"先连后断"的策略。
- **DBSCAN**：基于局部密度可达性进行聚类。通过 ε（邻域半径）和 MinPts（最少点数）两个参数定义密度。属于"由点及面"的策略。

MST聚类的优势在于能自然地处理密度不均匀的数据（通过局部不一致性度量），而DBSCAN在密度变化剧烈时可能需要多次调参。但DBSCAN能直接识别噪声点（不属于任何簇的点），MST聚类则需要额外的后处理。

结构聚类广泛应用于：
- 社交网络社区发现
- 图像分割
- 地理空间数据分析
- 生物信息学中的蛋白质交互网络分析

## 结构聚类算法的java实现

下面实现基于最小生成树的结构聚类，使用Kruskal算法构建MST，然后通过删边策略进行聚类：

```
import java.util.*;

public class MSTClustering {

    // 图的边
    static class Edge implements Comparable<Edge> {
        int from, to;
        double weight;

        Edge(int from, int to, double weight) {
            this.from = from; this.to = to; this.weight = weight;
        }

        @Override
        public int compareTo(Edge other) {
            return Double.compare(this.weight, other.weight);
        }

        @Override
        public String toString() {
            return "(" + from + "-" + to + ", " + String.format("%.4f", weight) + ")";
        }
    }

    // 并查集（Union-Find）
    static class UnionFind {
        int[] parent, rank;

        UnionFind(int n) {
            parent = new int[n];
            rank = new int[n];
            for (int i = 0; i < n; i++) parent[i] = i;
        }

        int find(int x) {
            if (parent[x] != x) parent[x] = find(parent[x]);
            return parent[x];
        }

        boolean union(int x, int y) {
            int px = find(x), py = find(y);
            if (px == py) return false;
            if (rank[px] < rank[py]) { int t = px; px = py; py = t; }
            parent[py] = px;
            if (rank[px] == rank[py]) rank[px]++;
            return true;
        }
    }

    private double[][] data;
    private List<Edge> mstEdges;

    public MSTClustering(double[][] data) {
        this.data = data;
    }

    private double distance(double[] a, double[] b) {
        double sum = 0;
        for (int i = 0; i < a.length; i++) sum += Math.pow(a[i] - b[i], 2);
        return Math.sqrt(sum);
    }

    // Kruskal算法构建最小生成树
    public List<Edge> buildMST() {
        int n = data.length;
        List<Edge> allEdges = new ArrayList<>();

        for (int i = 0; i < n; i++)
            for (int j = i + 1; j < n; j++)
                allEdges.add(new Edge(i, j, distance(data[i], data[j])));

        Collections.sort(allEdges);

        UnionFind uf = new UnionFind(n);
        mstEdges = new ArrayList<>();

        for (Edge edge : allEdges) {
            if (uf.union(edge.from, edge.to)) {
                mstEdges.add(edge);
                if (mstEdges.size() == n - 1) break;
            }
        }
        return mstEdges;
    }

    // 方法1：删除k-1条最长边，得到k个簇
    public int[] clusterByRemovingLongestEdges(int k) {
        if (mstEdges == null) buildMST();

        List<Edge> sortedEdges = new ArrayList<>(mstEdges);
        sortedEdges.sort((a, b) -> Double.compare(b.weight, a.weight));

        // 删除最长的k-1条边
        Set<Edge> removedEdges = new HashSet<>();
        System.out.println("删除的边:");
        for (int i = 0; i < k - 1 && i < sortedEdges.size(); i++) {
            removedEdges.add(sortedEdges.get(i));
            System.out.println("  " + sortedEdges.get(i));
        }

        // 用剩余边重建连通分量
        int n = data.length;
        UnionFind uf = new UnionFind(n);
        for (Edge edge : mstEdges)
            if (!removedEdges.contains(edge))
                uf.union(edge.from, edge.to);

        // 分配聚类标签
        Map<Integer, Integer> rootToLabel = new HashMap<>();
        int[] labels = new int[n];
        int nextLabel = 0;
        for (int i = 0; i < n; i++) {
            int root = uf.find(i);
            if (!rootToLabel.containsKey(root))
                rootToLabel.put(root, nextLabel++);
            labels[i] = rootToLabel.get(root);
        }
        return labels;
    }

    // 方法2：基于边权重的不一致性度量（inconsistency）
    public int[] clusterByInconsistency(double threshold) {
        if (mstEdges == null) buildMST();

        // 计算每条边的不一致性：与相邻边权重的比值
        double meanWeight = 0;
        for (Edge e : mstEdges) meanWeight += e.weight;
        meanWeight /= mstEdges.size();

        double stdWeight = 0;
        for (Edge e : mstEdges) stdWeight += Math.pow(e.weight - meanWeight, 2);
        stdWeight = Math.sqrt(stdWeight / mstEdges.size());

        Set<Edge> removedEdges = new HashSet<>();
        System.out.println("不一致性删边 (阈值=" + threshold + "):");
        for (Edge edge : mstEdges) {
            double inconsistency = (edge.weight - meanWeight) / (stdWeight + 1e-10);
            if (inconsistency > threshold) {
                removedEdges.add(edge);
                System.out.println("  " + edge + " (不一致性=" + String.format("%.4f", inconsistency) + ")");
            }
        }

        int n = data.length;
        UnionFind uf = new UnionFind(n);
        for (Edge edge : mstEdges)
            if (!removedEdges.contains(edge))
                uf.union(edge.from, edge.to);

        Map<Integer, Integer> rootToLabel = new HashMap<>();
        int[] labels = new int[n];
        int nextLabel = 0;
        for (int i = 0; i < n; i++) {
            int root = uf.find(i);
            if (!rootToLabel.containsKey(root))
                rootToLabel.put(root, nextLabel++);
            labels[i] = rootToLabel.get(root);
        }
        return labels;
    }

    public static void main(String[] args) {
        // 包含3个明显簇的数据
        double[][] data = {
            {1.0, 1.0}, {1.5, 1.5}, {1.2, 1.3}, {0.8, 1.2}, {1.3, 0.9},
            {5.0, 5.0}, {5.5, 5.5}, {5.2, 5.3}, {4.8, 5.2}, {5.3, 4.8},
            {9.0, 1.0}, {9.5, 1.5}, {9.2, 1.3}, {8.8, 0.8}, {9.3, 1.1}
        };

        MSTClustering mstClust = new MSTClustering(data);

        System.out.println("=== 最小生成树构建 ===");
        List<Edge> mst = mstClust.buildMST();
        System.out.println("MST边:");
        for (Edge e : mst) System.out.println("  " + e);
        double totalWeight = mst.stream().mapToDouble(e -> e.weight).sum();
        System.out.println("MST总权重: " + String.format("%.4f", totalWeight));

        System.out.println("\n=== 方法1：删除最长边聚类 (K=3) ===");
        int[] labels1 = mstClust.clusterByRemovingLongestEdges(3);
        System.out.println("聚类结果:");
        for (int i = 0; i < data.length; i++)
            System.out.println("  样本" + i + " " + Arrays.toString(data[i]) + " -> 簇" + labels1[i]);

        System.out.println("\n=== 方法2：不一致性度量聚类 ===");
        int[] labels2 = mstClust.clusterByInconsistency(1.0);
        System.out.println("聚类结果:");
        for (int i = 0; i < data.length; i++)
            System.out.println("  样本" + i + " " + Arrays.toString(data[i]) + " -> 簇" + labels2[i]);
    }
}
```

下面实现Prim算法构建MST，并使用基于局部密度的不一致性度量进行更智能的边删除：

```
import java.util.*;

public class PrimMSTClustering {

    private double[][] data;
    private int[] mstParent;
    private double[] mstKey;

    public PrimMSTClustering(double[][] data) {
        this.data = data;
    }

    private double distance(double[] a, double[] b) {
        double sum = 0;
        for (int i = 0; i < a.length; i++) sum += Math.pow(a[i] - b[i], 2);
        return Math.sqrt(sum);
    }

    public void buildMSTPrim() {
        int n = data.length;
        boolean[] inMST = new boolean[n];
        mstKey = new double[n];
        mstParent = new int[n];

        Arrays.fill(mstKey, Double.MAX_VALUE);
        Arrays.fill(mstParent, -1);
        mstKey[0] = 0;

        for (int count = 0; count < n; count++) {
            int u = -1;
            double minKey = Double.MAX_VALUE;
            for (int v = 0; v < n; v++)
                if (!inMST[v] && mstKey[v] < minKey) { minKey = mstKey[v]; u = v; }

            inMST[u] = true;

            for (int v = 0; v < n; v++) {
                if (!inMST[v]) {
                    double dist = distance(data[u], data[v]);
                    if (dist < mstKey[v]) { mstKey[v] = dist; mstParent[v] = u; }
                }
            }
        }
    }

    private Map<Integer, List<int[]>> buildAdjacency() {
        Map<Integer, List<int[]>> adj = new HashMap<>();
        int n = data.length;
        for (int i = 0; i < n; i++) adj.put(i, new ArrayList<>());

        for (int i = 1; i < n; i++) {
            adj.get(i).add(new int[]{mstParent[i], i});
            adj.get(mstParent[i]).add(new int[]{i, mstParent[i]});
        }
        return adj;
    }

    private List<Double> getNeighborWeights(int node, Map<Integer, List<int[]>> adj, int depth) {
        List<Double> weights = new ArrayList<>();
        Set<Integer> visited = new HashSet<>();
        Queue<int[]> queue = new LinkedList<>();
        queue.add(new int[]{node, 0});
        visited.add(node);

        while (!queue.isEmpty()) {
            int[] curr = queue.poll();
            int currNode = curr[0], currDepth = curr[1];
            if (currDepth >= depth) continue;

            for (int[] neighbor : adj.get(currNode)) {
                int next = neighbor[0];
                if (!visited.contains(next)) {
                    visited.add(next);
                    double w = distance(data[currNode], data[next]);
                    weights.add(w);
                    queue.add(new int[]{next, currDepth + 1});
                }
            }
        }
        return weights;
    }

    public int[] clusterByLocalDensity(int neighborDepth, double sensitivity) {
        if (mstParent == null) buildMSTPrim();

        int n = data.length;
        Map<Integer, List<int[]>> adj = buildAdjacency();

        Set<Integer> removedEdges = new HashSet<>();
        System.out.println("基于局部密度的不一致性分析 (邻域深度=" + neighborDepth
            + ", 敏感度=" + sensitivity + "):");

        for (int i = 1; i < n; i++) {
            double edgeWeight = distance(data[i], data[mstParent[i]]);
            List<Double> neighborWeights = new ArrayList<>();
            neighborWeights.addAll(getNeighborWeights(i, adj, neighborDepth));
            neighborWeights.addAll(getNeighborWeights(mstParent[i], adj, neighborDepth));

            if (neighborWeights.size() < 2) continue;

            double localMean = neighborWeights.stream().mapToDouble(w -> w).average().orElse(0);
            double localStd = Math.sqrt(neighborWeights.stream()
                .mapToDouble(w -> Math.pow(w - localMean, 2)).average().orElse(0));

            double zScore = (edgeWeight - localMean) / (localStd + 1e-10);

            if (zScore > sensitivity) {
                removedEdges.add(i);
                System.out.println("  删除边 (" + mstParent[i] + "-" + i
                    + ", w=" + String.format("%.4f", edgeWeight)
                    + ") 局部z-score=" + String.format("%.4f", zScore));
            }
        }

        int[] parent = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
        for (int i = 1; i < n; i++) {
            if (!removedEdges.contains(i)) {
                int px = find(parent, i), py = find(parent, mstParent[i]);
                if (px != py) parent[px] = py;
            }
        }

        Map<Integer, Integer> rootToLabel = new HashMap<>();
        int[] labels = new int[n];
        int nextLabel = 0;
        for (int i = 0; i < n; i++) {
            int root = find(parent, i);
            if (!rootToLabel.containsKey(root)) rootToLabel.put(root, nextLabel++);
            labels[i] = rootToLabel.get(root);
        }
        return labels;
    }

    private int find(int[] parent, int x) {
        while (parent[x] != x) { parent[x] = parent[parent[x]]; x = parent[x]; }
        return x;
    }

    public static void main(String[] args) {
        double[][] data = {
            {1.0, 1.0}, {1.5, 1.5}, {1.2, 1.3}, {0.8, 1.2}, {1.3, 0.9},
            {5.0, 5.0}, {5.5, 5.5}, {5.2, 5.3}, {4.8, 5.2}, {5.3, 4.8},
            {9.0, 1.0}, {9.5, 1.5}, {9.2, 1.3}, {8.8, 0.8}, {9.3, 1.1}
        };

        System.out.println("=== Prim算法构建MST + 基于局部密度的聚类 ===\n");

        PrimMSTClustering prim = new PrimMSTClustering(data);
        prim.buildMSTPrim();

        System.out.println("Prim MST 边:");
        for (int i = 1; i < data.length; i++) {
            double w = 0;
            for (int d = 0; d < data[0].length; d++)
                w += Math.pow(data[i][d] - data[prim.mstParent[i]][d], 2);
            w = Math.sqrt(w);
            System.out.println("  " + prim.mstParent[i] + " -- " + i
                + " (权重: " + String.format("%.4f", w) + ")");
        }

        System.out.println();
        int[] labels = prim.clusterByLocalDensity(2, 1.5);

        System.out.println("\n聚类结果:");
        Map<Integer, List<Integer>> clusters = new TreeMap<>();
        for (int i = 0; i < labels.length; i++) {
            clusters.computeIfAbsent(labels[i], k -> new ArrayList<>()).add(i);
        }
        for (Map.Entry<Integer, List<Integer>> entry : clusters.entrySet()) {
            System.out.println("  簇 " + entry.getKey() + ": " + entry.getValue());
        }
    }
}
```

下面实现轮廓系数（Silhouette Score）用于评估聚类质量，并给出一个社交网络社区发现的实际应用示例：

```
import java.util.*;

public class MSTClusteringAdvanced {

    static double distance(double[] a, double[] b) {
        double sum = 0;
        for (int i = 0; i < a.length; i++) sum += Math.pow(a[i] - b[i], 2);
        return Math.sqrt(sum);
    }

    static class UnionFind {
        int[] parent, rank;
        UnionFind(int n) {
            parent = new int[n]; rank = new int[n];
            for (int i = 0; i < n; i++) parent[i] = i;
        }
        int find(int x) {
            if (parent[x] != x) parent[x] = find(parent[x]);
            return parent[x];
        }
        boolean union(int x, int y) {
            int px = find(x), py = find(y);
            if (px == py) return false;
            if (rank[px] < rank[py]) { int t = px; px = py; py = t; }
            parent[py] = px;
            if (rank[px] == rank[py]) rank[px]++;
            return true;
        }
    }

    static double silhouetteScore(double[][] data, int[] labels) {
        int n = data.length;
        int numClusters = 0;
        for (int l : labels) numClusters = Math.max(numClusters, l + 1);

        if (numClusters <= 1) return 0;

        double totalScore = 0;
        for (int i = 0; i < n; i++) {
            double a_i = 0;
            int sameCount = 0;
            double[] otherDist = new double[numClusters];
            int[] otherCount = new int[numClusters];

            for (int j = 0; j < n; j++) {
                if (i == j) continue;
                double d = distance(data[i], data[j]);
                if (labels[j] == labels[i]) { a_i += d; sameCount++; }
                else { otherDist[labels[j]] += d; otherCount[labels[j]]++; }
            }
            a_i = (sameCount > 0) ? a_i / sameCount : 0;

            double b_i = Double.MAX_VALUE;
            for (int c = 0; c < numClusters; c++) {
                if (c != labels[i] && otherCount[c] > 0)
                    b_i = Math.min(b_i, otherDist[c] / otherCount[c]);
            }
            if (b_i == Double.MAX_VALUE) b_i = 0;

            double s_i = (Math.max(a_i, b_i) > 0) ? (b_i - a_i) / Math.max(a_i, b_i) : 0;
            totalScore += s_i;
        }
        return totalScore / n;
    }

    static int[][] buildMST(double[][] data) {
        int n = data.length;
        List<int[]> edges = new ArrayList<>();
        List<Double> weights = new ArrayList<>();

        List<double[]> allEdges = new ArrayList<>();
        for (int i = 0; i < n; i++)
            for (int j = i + 1; j < n; j++)
                allEdges.add(new double[]{i, j, distance(data[i], data[j])});
        allEdges.sort((a, b) -> Double.compare(a[2], b[2]));

        UnionFind uf = new UnionFind(n);
        for (double[] e : allEdges) {
            int u = (int) e[0], v = (int) e[1];
            if (uf.union(u, v)) {
                edges.add(new int[]{u, v});
                weights.add(e[2]);
                if (edges.size() == n - 1) break;
            }
        }

        int[][] result = new int[edges.size()][2];
        for (int i = 0; i < edges.size(); i++) result[i] = edges.get(i);
        return result;
    }

    static int[] mstCluster(double[][] data, int k) {
        int n = data.length;
        List<double[]> allEdges = new ArrayList<>();
        for (int i = 0; i < n; i++)
            for (int j = i + 1; j < n; j++)
                allEdges.add(new double[]{i, j, distance(data[i], data[j])});
        allEdges.sort((a, b) -> Double.compare(a[2], b[2]));

        UnionFind uf = new UnionFind(n);
        List<double[]> mst = new ArrayList<>();
        for (double[] e : allEdges) {
            if (uf.union((int) e[0], (int) e[1])) {
                mst.add(e);
                if (mst.size() == n - 1) break;
            }
        }
        mst.sort((a, b) -> Double.compare(b[2], a[2]));

        UnionFind uf2 = new UnionFind(n);
        for (int i = k - 1; i < mst.size(); i++)
            uf2.union((int) mst.get(i)[0], (int) mst.get(i)[1]);

        Map<Integer, Integer> rootToLabel = new HashMap<>();
        int[] labels = new int[n];
        int nextLabel = 0;
        for (int i = 0; i < n; i++) {
            int root = uf2.find(i);
            if (!rootToLabel.containsKey(root)) rootToLabel.put(root, nextLabel++);
            labels[i] = rootToLabel.get(root);
        }
        return labels;
    }

    public static void main(String[] args) {
        System.out.println("=== 轮廓系数评估 + 社交网络社区发现 ===\n");

        double[][] data = {
            {1.0, 1.0}, {1.5, 1.5}, {1.2, 1.3}, {0.8, 1.2}, {1.3, 0.9},
            {5.0, 5.0}, {5.5, 5.5}, {5.2, 5.3}, {4.8, 5.2}, {5.3, 4.8},
            {9.0, 1.0}, {9.5, 1.5}, {9.2, 1.3}, {8.8, 0.8}, {9.3, 1.1}
        };

        System.out.println("--- 轮廓系数选择最优聚类数 ---\n");
        System.out.println(String.format("%-8s %-15s", "聚类数K", "轮廓系数"));
        System.out.println("-".repeat(23));

        int bestK = 2;
        double bestScore = -1;
        for (int k = 2; k <= 6; k++) {
            int[] labels = mstCluster(data, k);
            double score = silhouetteScore(data, labels);
            System.out.println(String.format("%-8d %-15.4f", k, score));
            if (score > bestScore) { bestScore = score; bestK = k; }
        }
        System.out.println("最优聚类数: K=" + bestK + " (轮廓系数=" + String.format("%.4f", bestScore) + ")\n");

        System.out.println("--- 社交网络社区发现示例 ---\n");

        String[] users = {"Alice", "Bob", "Carol", "Dave", "Eve",
                          "Frank", "Grace", "Hank", "Ivy", "Jack"};
        double[][] socialFeatures = {
            {1.0, 8.0, 5.0},
            {1.5, 7.5, 4.8},
            {2.0, 7.0, 5.2},
            {8.0, 2.0, 1.0},
            {7.5, 1.5, 1.2},
            {8.5, 2.5, 0.8},
            {7.8, 1.8, 1.1},
            {4.0, 4.0, 8.0},
            {4.5, 4.5, 7.5},
            {3.5, 3.8, 8.2}
        };

        System.out.println("用户特征向量 [发帖频率, 互动频率, 兴趣相似度]:");
        for (int i = 0; i < users.length; i++)
            System.out.println("  " + String.format("%-8s", users[i]) + ": " + Arrays.toString(socialFeatures[i]));

        int[] communityLabels = mstCluster(socialFeatures, 3);
        double score = silhouetteScore(socialFeatures, communityLabels);

        System.out.println("\n发现的社区 (轮廓系数=" + String.format("%.4f", score) + "):");
        Map<Integer, List<String>> communities = new TreeMap<>();
        for (int i = 0; i < users.length; i++)
            communities.computeIfAbsent(communityLabels[i], k -> new ArrayList<>()).add(users[i]);

        for (Map.Entry<Integer, List<String>> entry : communities.entrySet())
            System.out.println("  社区 " + entry.getKey() + ": " + entry.getValue());

        System.out.println("\n分析：基于MST的社区发现能识别出具有相似行为模式的用户群体，");
        System.out.println("不同社区的用户在特征空间中形成自然分离的簇。");
    }
}
```

## 总结

基于最小生成树的结构聚类算法是结构模式识别的经典方法，其主要特点：

1. **能发现任意形状的簇**：不像K-Means那样假设簇是球形的，MST聚类可以发现链状、环状等任意形状的聚类
2. **基于图论的清晰框架**：利用图的连通性来定义聚类，数学基础严谨
3. **层次化结构**：删除不同数量的边可以得到不同粒度的聚类结果
4. **无需指定初始中心**：避免了K-Means的初始化问题

其主要局限性：
- 时间复杂度较高：构建完全图需要 O(n²)，排序需要 O(n²logn)
- 对噪声敏感：噪声点可能导致异常的长边连接
- 删边策略需要人工参与：如何选择合适的阈值或删边数量

## 图聚类方法对比

| 特性 | MST聚类 | DBSCAN | 谱聚类 | Louvain社区检测 |
|------|---------|--------|--------|-----------------|
| **核心思想** | MST删边分割 | 密度可达性 | 图拉普拉斯谱分解 | 模块度最大化 |
| **簇形状** | 任意形状 | 任意形状 | 任意形状 | 网络社区结构 |
| **输入参数** | 聚类数k或阈值 | ε, MinPts | 聚类数k | 无（自动确定） |
| **时间复杂度** | O(n²logn) | O(n logn) ~ O(n²) | O(n³) | O(n log n) |
| **噪声处理** | 无内置机制 | 自动标记噪声点 | 无内置机制 | 无 |
| **密度不均匀** | 局部不一致性可处理 | 困难（全局ε） | 能处理 | 能处理 |
| **可扩展性** | 中等 | 好 | 差（大规模数据） | 非常好 |
| **确定性** | 确定性结果 | 确定性结果 | 随机初始化影响 | 可能不确定 |
| **层次结构** | 天然支持 | 不支持 | 不支持 | 支持多层次 |
| **适用数据** | 空间数据、几何数据 | 空间数据 | 流形数据 | 网络/图数据 |

**选择建议**：
- 需要发现任意形状簇、数据量中等 → MST聚类
- 数据中有明显噪声 → DBSCAN
- 数据分布在复杂流形上 → 谱聚类
- 分析社交网络、引用网络等图结构 → Louvain或其他社区检测
- 需要层次化聚类结果 → MST聚类（配合不同删边数量）

在实际应用中，MST聚类常与密度估计、噪声过滤等技术结合使用，以提高其在复杂数据上的鲁棒性。
