本故事纯属虚构，如有雷同，纯属巧合。

>   中国武术强调内练一口气、外练筋骨皮。
>
> 《易筋经》是锻炼筋膜和骨膜的，属于“外练筋骨皮”。
>
> 《洗髓经》是锻炼内功的，属于“内练一口气”，即丹田混元真气。。
>
> 《易筋经》、《洗髓经》一外、一内，缺一不可。  

**故事背景**

​	江湖传说中，总会有一些盖世的武功秘籍，学完他就可以打通任督二穴，打遍天下无敌手了，如《易筋经》、《洗髓经》，作为程序猿的小白也想找到 Java 的《易筋经》、《洗髓经》。那我们就来探讨一二吧。

**Java 语言的《易筋经》**

​	刚开始的时候，小白在《Java核心技术》(上下两卷)、《Java编程思想》中两本书中举棋不定，两本书都略显枯燥学术，作为入门都很难坚持完完整整的看完一遍。

![img](image/dbc89e91fb6b4cbba6af79ea7d453967.jfif)

![img](image/8f4f895e421744bd859c78405d543923.jfif)

​	随着进入工作经验的增加，觉得《重构:改善既有代码的设计》、《**Effective Java**》这些书更适合，特别是后者，不厚，更符合实际工作。

![img](image/35dd7671d9ef491fa22e92e2c52cff73.jfif)



![img](image/52a565bba2934c8e8f9bd957210d8e0f.jfif)

​	后来读到 《Java解惑》(Java puzzle) 后，他以为我找到了真正的《易筋经》。但我最后发现，真正的《易筋经》不是《 Java 解惑》，而是 JLS。

![img](image/6cf5a082f13545658817481fe2fd26cd.jfif)

何为 JLS？ **The Java Language Specification** 即 **《Java 语言规范》**。

​	由 Java 技术的发明者编写，是 Java 编程语言的权威性技术指南。全面、准确而详细地讨论了 Java 编程语言，是 Java 语言最新版本的规范。从 Java 最基础的文法、类型、变量到高级特性 lambda 表达式、线程与锁等进行了详尽阐述。该规范针对每一项语言特性都给出了代表性的示例程序，以帮助读者更容易地理解和掌握这些特性。通过阅读本规范，可以全面系统地了解 Java 语言的各项特性，为了充分利用这些特性来编写出更加高效、简洁的 Java 程序提供帮助。

最新版本的 JLS 为 Java SE 13，发布于2019年9月，提供了 html 和 pdf 版本，可以到https://docs.oracle.com/javase/specs/上面查看或者下载。

目前能搜索到的中文版本是基于 JAVA SE 8 的。

![img](image/9d1be8fa966d4967966c2756a7dc0dec.jfif)



**Java语言的《洗髓经》**

The Java Virtual Machine Specification

![img](image/5bf1e43661fb4e818a7e31f33ca594b9.jfif)

​	完整而准确地阐释了 Java 虚拟机各方面的细节，围绕 Java 虚拟机整体架构、编译器、class 文件格式、加载、链接与初始化、指令集等核心主题对 Java 虚拟机进行全面而深入的分析，深刻揭示 Java 虚拟机的工作原理。

### 总结

身为 Java 程序员的你可能正在思考平时应该看一些什么书籍来提高自己的编码能力，市面上的书籍汗牛充栋，追本溯源，JLS(Java 语言规范) 作为 Java 语言的本源，在工作一段时间后，可以回过头来仔细研读一番，一方面从以往的坑中找到相应的问题，另一方面可以由点到面的完善 Java 语法上的知识，查缺补漏。JVMS 则是 Java 运行的奇经八脉，想要打通任督二穴，不可或缺！
