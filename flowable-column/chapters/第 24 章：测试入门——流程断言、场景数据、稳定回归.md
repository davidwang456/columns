## 1）项目背景

星云 CI 要求：**每次 MR** 跑通覆盖核心路径的 **流程集成测试**；研发本地需要 **秒级反馈**。**本章聚焦**：测试金字塔（纯 Delegate 单测、流程集成测、REST 契约测）、**H2 vs PG**、**并发测试数据隔离**、以及 **async/定时** 的可重复验证策略。

## 2）项目设计（小胖 × 小白 × 大师）

> **角色分工**：**小胖**（爱吃爱玩、不求甚解）用生活化、口语化的方式把话题「开球」，先把问题问出来；**小白**（喜静、喜深入）负责追问原理、边界、风险与可比方案；**大师**（资深技术 Leader）把业务约束与技术选型说透，善打比方、由浅入深。

**小胖**：只 Mockito Delegate 不够吗？

**大师**：不够；BPMN **连线错误**只有跑起来才发现。

**小白**：用内存 H2 可以吗？

**大师**：开发首选；上线前至少在 CI 加一轮 **PG 容器** 测，防方言差异。

**小胖**：测试数据脏？

**大师**：**@Transactional rollback** 或 **随机 schema** 或 **@DirtiesContext** 慎用。

**小白**：异步怎么等？

**大师**：`Awaitility`、轮询历史、对 Timer 用 **短周期**。

## 2.1）场景深化：场景矩阵与「变更影响面」标签

**星云**把每条流程测试用例打上标签：`P0-资金`、`P1-通知`、`P2-报表`。当某 PR 只改通知监听，CI 可先跑 **P1+P2 子集**加速；触及网关金额条件则 **全量 P0**。这是控制 30 分钟构建时间不爆炸的技巧。

## 3）项目实战（主代码片段）

### 3.1 Spring Boot 集成测骨架

```java
@SpringBootTest(properties = "spring.profiles.active=test")
class ProcurementProcessIT {
  @Autowired RuntimeService runtimeService;
  @Autowired TaskService taskService;
  @Autowired HistoryService historyService;

  @Test
  void shouldCompleteHappyPath() {
    var pi = runtimeService.startProcessInstanceByKey(
        "procurement", "BK-IT-"+UUID.randomUUID(), Map.of("amount", 8000L));
    var task = taskService.createTaskQuery().processInstanceId(pi.getId()).singleResult();
    taskService.complete(task.getId(), Map.of("approved", true));
    var hip = historyService.createHistoricProcessInstanceQuery()
        .processInstanceId(pi.getId()).singleResult();
    assertThat(hip.getEndTime()).isNotNull();
  }
}
```

## 4）项目总结

### 优点

- **早发现问题**；  
- 新人可读测试学流程。

### 缺点

- CI 时间成本；  
- **flaky** 需治理。

### 典型使用场景

- 核心审批、计价、集成外部 mock。

### 注意事项

- **时钟**与 **异步**；  
- **外部服务** stub。

### 反例

依赖「上次手造库里的用户」导致本地永远绿、CI 永远红。**纠正**：**@Sql** 或 API 造数。

### 常见踩坑

| 现象 | 根因 | 处理 |
|------|------|------|
| flaky | async | await |
| 脏数据 | 无隔离 | trx rollback |
| 慢 | 冷启动大 | 拆分测 |

### 给测试一句话

**场景表**：分支 × 变量 × 角色。

### 给运维一句话

CI **制品**附带 **BPMN 校验摘要** hash。

## 附录：星云案例深潜——「绿构建但生产线崩」的标签故事

CI 常年跳过 @Slow 标记的集成测试，一次网关变更在生产暴露。之后策略是：**P0 永不标记 Slow**；Slow 只能是非关键路径。测试标签与 **发布门禁**必须同源治理。
