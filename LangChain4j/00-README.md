# LangChain4j 由浅入深（38 章）

本目录为配套**独立 Markdown 章节**，与本地仓库 [`langchain4j`](../../langchain4j) 及 [`langchain4j-examples`](../../langchain4j-examples) 中的示例代码对应。每章建议阅读时间 45～90 分钟（含动手）。

## 阅读约定

- 每章包含：**项目背景**、**大师与小白对话**、**项目实战（主代码片段）**、**项目总结**（优缺点、场景、注意事项、踩坑）。
- 文末附有 **测试/运维检查清单**（80～120 字量级，可改为工单模板）。
- 详细合并逻辑见 [CHAPTER_MERGE_LOG.md](CHAPTER_MERGE_LOG.md)。
