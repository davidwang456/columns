## 建议拆分（示例）

| 职责 | Redis 能力 | 备注 |
|------|------------|------|
| 商品详情缓存 | String / Hash + TTL | 热点 key 走 [卷七-25](卷七-25-可观测与HOTKEYS.md) |
| 用户短期兴趣 | ZSet（带时间衰减 score）或 Stream 聚合 | 与团队约定 |
| 向量召回 | Vector set `VADD`/`VSIM` | 嵌入离线/准实时更新 |
| 点击流 | Stream；生产端 **IDMP** 防重 | 见 [卷三-13](卷三-13-Stream消费组与XADD幂等.md) |
| 订单事件 | Stream 另一 key 或业务 Kafka | **Redis 不是万能 MQ** |
