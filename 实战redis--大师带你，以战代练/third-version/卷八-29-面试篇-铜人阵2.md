## 铜人阵 2.0：Redis 8.x 必答增量

**大师**：旧阵考缓存、结构、管道、一致性；新阵加 **Hash TTL、Stream IDMP、lrm、HOTKEYS、Vector、OSS/云边界**。

---

### 1. Hash 字段过期 vs Key TTL

**答**：**7.4+** 字段级过期；**8.0+** `HSETEX`；避免整 key 误杀。复制/持久化语义读发行说明。

### 2. `XADD` 幂等（8.6）

**答**：**`IDMP`/`IDMPAUTO`** + **`*` 自动 ID**；防**写重试双条**；**消费幂等**另做。

### 3. `volatile-lrm` / `allkeys-lrm`

**答**：**最近最少修改**向淘汰；与 LRU/LFU 场景差异。

### 4. `HOTKEYS`（8.6）

**答**：热点发现；注意**开销**与**环境**。

### 5. Vector set 与 RAG

**答**：**应用侧嵌入**，Redis **`VSIM`**；**Query Engine** 与 **Vector set** 分清楚。

### 6. JSON vs 字符串 JSON

**答**：迁移路径；索引与查询在 **Query Engine** 章节。

### 7. `BUILD_WITH_MODULES`

**答**：Bloom 等 **\*** 能力；`MODULE LIST` 自检。

### 8. 文档与二进制不一致

**答**：以 **`COMMAND` + 版本**为准；**诚实**比背题高分。
