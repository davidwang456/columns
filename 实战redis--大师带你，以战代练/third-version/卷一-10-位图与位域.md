## 位图与位域：日活与 RBAC 的「针尖功」

**大师**：**Bitmap** 用 `SETBIT` 记「某天某用户是否登录」——省内存、怕稀疏。 **Bitfield** 则在同一串里塞**多个有符号/无符号整数位段**，适合 **RBAC 权限位**、小型计数。

**小白**：Bitmap 和 Set 谁省？

**大师**：**极稠密**用 Bitmap；**稀疏**用 Set 或 Bloom（模块）。**错选型比错命令更费钱**。
