| 分类           | 核心职责    | 代表性开源框架                                                                            |
|:------------ |:------- |:---------------------------------------------------------------------------------- |
| **数据采集与传输**  | 收集和搬运数据 | `Kafka`, `Flume`, `Sqoop`, `Pulsar`，`DATAX`                                        |
| **数据存储**     | 存放海量数据  | `HDFS`, `HBase`, `Kudu`, `Alluxio`                                                 |
| **资源调度与管理**  | 分配计算资源  | `YARN`, `Mesos`, `Kubernetes (K8s)`                                                |
| **数据计算与处理**  | 加工和分析数据 | **批处理**: `Spark`, `Hive`, `MapReduce` **流处理**: `Flink`, `Spark Streaming`, `Storm` |
| **数据查询与分析**  | 快速交互式查询 | `Presto (Trino)`, `Impala`, `ClickHouse`, `Druid`, `Kylin`                         |
| **任务调度与工作流** | 编排数据任务  | `Airflow`, `DolphinScheduler`, `Azkaban`, `Oozie`                                  |
